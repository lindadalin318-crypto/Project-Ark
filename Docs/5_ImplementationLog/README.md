# Implementation Log — Project Ark

## 职责

`5_ImplementationLog/` 负责记录：

- 改了什么
- 为什么改
- 使用了什么技术

它不替代设计文档、技术规范或项目计划。

## 文件结构

- [`ImplementationLog.md`](./ImplementationLog.md): 历史总账，保留旧时代连续记录
- [`ImplementationLog_2026-04.md`](./ImplementationLog_2026-04.md): 新体系启动后的月度日志入口

## 维护规则

- 旧历史不强拆，继续保留在 `ImplementationLog.md`
- 新体系从 `2026-04` 起按月份新增日志文件
- 每次有文件改动，都必须在当前回合补日志
