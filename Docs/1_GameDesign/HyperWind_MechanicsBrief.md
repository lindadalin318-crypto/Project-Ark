# 超旋大气星 · 机制简报（HyperWind Mechanics Brief）

> **版本**：v0.1 (2026-05-15)
> **定位**：星球3 候选方案（Heart-Wonder = 超旋大气）的机制盘点稿
> **目的**：在进入 Planet Bible / Feature Requirements 详写之前，先把所有候选机制压成一页可读完的清单，**用于挑选 MVP 切片**
> **状态**：未PK定锚（与 02倒影星 / 04折叠腹地星 同处星球3候选位）
> **基线奇景**：`Ark_WonderCandidatePool.md` §1.7 超旋大气
> **设计原则**：所有机制挂载到一张全局风向场（每像素含风向+风速），衍生玩法

---

## 0. 核心设定一句话

**整颗星球被一道横切赤道的高速气流环带统治，所有移动、射击、敌人行为都被风向场调制。**

- 风带形态：环状 / 脉动（45-90秒强弱循环）/ 永不停歇
- 玩家学习曲线：恐惧（被裹挟）→ 理解（读风）→ 驾驭（用风）

---

## 1. 全局星球机制（Global / Wonder Rhythm）

| ID | 机制 | 一句话定义 | 玩家感受 |
|---|---|---|---|
| G1 | **风向场** | 全局矢量场，每像素有风向+风速；所有飞船/子弹/敌人/粒子从此读数 | 一切运动都"歪"一点 |
| G2 | **风相位脉动** | 45-90秒一次的强弱周期，全星球同步 | 永远在等下一次"风暴呼吸" |
| G3 | **三阶段预警** | 沙浪先动(5s)→远处轰鸣(3s)→风速暴涨(0s) | 给玩家读节律的窗口 |
| G4 | **强相位套件** | 子弹偏折x2、敌人借风、风骑兵无敌、能量结晶刷新 | 高威胁高奖励 |
| G5 | **弱相位套件** | 风影区扩大、风眼可见、敌人显形 | 输出/通行/解谜窗口 |
| G6 | **超级脉动**（30min一次） | 全图风向重写，强制玩家适应新地形 | 史诗级事件 |
| G7 | **静风事件**（罕见） | 1分钟全星球静风，所有规则消失 | 反差型震撼 |
| G8 | **五大风区** | 风眼 / 乱流带 / 顺风走廊 / 背风影区 / 极地静风 | 同一星球的5种"次气候" |

## 2. 移动机制（Movement / 飞船 × 风场）

| ID | 机制 | 一句话定义 | 战术价值 |
|---|---|---|---|
| M1 | **风矢叠加** | 飞船实际速度 = 推力矢量 + 风矢量 | 0教学，最基础 |
| M2 | **风滑流** | 紧贴顺风带边缘飞行获得加速 + 省燃料 | 老手专属技巧 |
| M3 | **风跳** | 短按冲刺穿过风带边界=被弹射，达到普通推进无法到达的位置 | 中期解锁的位移技 |
| M4 | **乱流眩晕** | 长时间停乱流带=操控延迟+随机偏移 | 阻止"乱流恋战" |
| M5 | **风眼真空** | 进风眼瞬间所有风力消失，操控切回标准物理 | 反差型奖励 |
| M6 | **风影锚点** | 山体/废墟背后的低风口袋，可主动停泊补给 | 鼓励读地形 |

## 3. 射击机制（Shooting / 子弹 × 风场）⭐ 双摇杆射击核心改造

| ID | 机制 | 一句话定义 | 战术深度 |
|---|---|---|---|
| S1 | **风偏弹道** | 子弹持续受风影响，远距离需"瞄左打右" | 基础改造 |
| S2 | **重弹绕飞** | 慢速大弹被风带走，可故意丢进顺风带打到背后敌人 | 战术爆点 |
| S3 | **激光直射** | 高速弹/激光不受风影响，但消耗高 | 武器选择有意义 |
| S4 | **散弹梳化** | 散射武器顺风时拉长、逆风时压扁、横风时变梳子状 | 一把武器三种用法 |
| S5 | **风刃大招** | 向风带"注入"自身能量，下一次脉动整条风带变可伤敌的"风刃带" | 把奇景武器化 |
| S6 | **声爆弹** | 朝逆风方向射击，弹头形成声爆环=范围伤害 | 鼓励反直觉决策 |
| S7 | **风蚀** | 子弹在乱流带久了会被磨损，伤害递减 | 限制远距离cheese |

## 6. 推荐 MVP 切片（3选1）

**MVP目标**：用一个5-10分钟的体验段落验证"超旋大气是否成立"——玩家能否感受到"风把游戏改变了"。

### 切片A · 风之走廊（最低成本，验证 M1+S1+G2）

- **场景**：单一房间，左侧是顺风走廊，右侧是逆风走廊，中间一面墙
- **机制覆盖**：G1风向场 + G2脉动 + M1风矢叠加 + S1风偏弹道
- **敌人**：3-5个 E1风骑兵
- **验证目标**：玩家在3分钟内是否能自然感受到"风的存在"且不别扭
- **资产成本**：⭐（最低，复用现有Room SO）
- **推荐优先级**：**P0**（先做）

### 切片B · 风眼挑战（验证奇景核心张力）

- **场景**：一片乱流带 + 中央风眼，风眼内悬浮目标物
- **机制覆盖**：G1+G2+G4+G5 + M1+M3+M5 + S1 + E1+E5
- **敌人**：风骑兵巡逻 + 迁徙群作为相位指示器
- **验证目标**：玩家是否能学会"在强相位躲、在弱相位冲"的节律
- **资产成本**：⭐⭐（中等，需新做风眼VFX）
- **推荐优先级**：P1

### 切片C · 风帆解谜（验证规则改写×关卡）

- **场景**：一座断风塔废墟，需激活L4断风器才能通过L1风门
- **机制覆盖**：G1+G2 + M1+M6 + S1 + L1+L4+L6 + E2
- **敌人**：风沙拟态在静风区显形
- **验证目标**：奇景是否能产生**新策略**而不只是新难度
- **资产成本**：⭐⭐⭐（最高，需新做机关交互）
- **推荐优先级**：P2

## 7. MVP 推荐路径

```
切片A（风之走廊）
   ↓ 验证基础风感
切片B（风眼挑战）
   ↓ 验证节律玩法
切片C（风帆解谜）
   ↓ 验证规则改写产生新策略
进入 Planet Bible 详写阶段
```

**MVP 第一步建议**：从切片A开始，**只实现 G1+G2+M1+S1+E1 五个机制**，约2周开发量，验证"风的手感"是否成立。如果手感成立，再扩展切片B/C。
