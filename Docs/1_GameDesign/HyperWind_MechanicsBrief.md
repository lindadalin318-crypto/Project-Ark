# 超旋大气星 · 机制简报（HyperWind Mechanics Brief）

> **版本**：v0.1 (2026-05-15)
> **定位**：星球3 候选方案（Heart-Wonder = 超旋大气）的机制盘点稿
> **目的**：在进入 Planet Bible / Feature Requirements 详写之前，先把所有候选机制压成一页可读完的清单，**用于挑选 MVP 切片**
> **状态**：未PK定锚（与 02倒影星 / 04折叠腹地星 同处星球3候选位）
> **基线奇景**：`Ark_WonderCandidatePool.md` §1.7 超旋大气
> **设计原则**：所有机制挂载到一张全局风向场（每像素含风向+风速），衍生玩法

---

## 0. 核心设定一句话

**整颗星球被一道横切赤道的高速气流环带统治，所有移动、射击、敌人行为都被风向场调制。**

- 风带形态：环状 / 脉动（45-90秒强弱循环）/ 永不停歇
- 玩家学习曲线：恐惧（被裹挟）→ 理解（读风）→ 驾驭（用风）

### 5.1 地面气旋族（Ground Cyclone Family）⭐

> **核心定义**：地面会周期性/事件性地生成局部气旋（涡旋点），它不是全局风带的一部分，而是**风向场上的局部奇点**——在它的影响半径内，所有子弹的运动逻辑被劫持。

#### 5.1.1 气旋生命周期（三阶段）

| 阶段 | 时长 | 视觉/反馈 | 子弹行为 |
|---|---|---|---|
| **生成期 Spawn** | 0.5s | 地面尘土卷起，提示圈出现 | 不影响子弹（玩家有反应窗口） |
| **吸吮期 Draw** | 3-5s | 气旋稳定旋转，可见涡流 | 进入半径的子弹被捕获，开始绕涡公转，**不消失、不衰减** |
| **释放期 Burst** | 0.3s | 气旋瞬间崩散，强光闪烁 | 累积的所有子弹一次性射出 |

#### 5.1.2 子弹捕获与绕飞机制

| ID | 机制 | 定义 |
|---|---|---|
| **L8-a** | **捕获判定** | 子弹进入气旋外环（影响半径）后被吸入轨道环（公转半径） |
| **L8-b** | **公转累积** | 多发子弹同时在轨道环上排队绕飞，互不碰撞，**视觉上形成可见弹环**（玩家可数清楚）|
| **L8-c** | **气旋强化（速度→伤害）** ⭐ | 被捕获的子弹在公转期间**持续加速**，释放时速度倍率 ×1.5~×3（按公转圈数累积）；命中"游戏内速度=伤害"基础规则，**伤害随之放大**。属性、归属保留 |
| **L8-d** | **存量上限** | 单个气旋最多容纳 N 发（建议 N=20），溢出的子弹直接消散，制造"不要塞太满"的张力 |

#### 5.1.3 释放方向（默认：方向继承；其余为未来扩展备选）

| 释放模式 | 行为 | 状态 |
|---|---|---|
| **方向继承** ⭐ 默认 | 朝玩家最后一次射击的方向集中射出 | **当前采用**——玩家可主动控制释放方向，与"延迟齐射"战术形成强配合 |
| 径向爆发 | 所有子弹朝公转切线方向四散（360°喷射） | 备选/未来扩展 |
| 聚焦齐射 | 所有子弹朝最近敌人锁定射出 | 备选/未来扩展 |
| 风向裹挟 | 释放方向跟随当前全局风向 | 备选/未来扩展 |

> **设计意图**：默认锁定"方向继承"是为了让 L8 的释放阶段成为**玩家技巧的延伸**而非自动效果——玩家"最后一次射击"是一次有意识的决策，气旋只是把这次决策放大。

#### 5.1.4 敌我归属规则（已确认）

- **吸入对象**：玩家弹 + 敌方弹**都会被吸**
- **归属保留**：**谁射的还是谁的**——玩家射的弹始终属于玩家（强化后命中敌人造成更高伤害），敌弹始终属于敌人（强化后命中玩家造成更高伤害）
- **释放方向**：所有被吸子弹（无论敌我）都按当前默认模式（方向继承）射出——这意味着**敌弹也会朝"玩家最后一次射击方向"飞**，可能反向打到玩家自己
- **第三方激活**：某些怪物（如风暴使者）可主动召唤气旋作为攻防装置

#### 5.1.5 气旋类型（按生成方式分3档）

| 类型 | 生成方式 | 出现频率 |
|---|---|---|
| **环境气旋** | 地图固定位置周期性自动生成（与风相位同步） | 高频（每相位多个） |
| **触发气旋** | 玩家或敌人主动召唤（消耗道具/技能） | 中频 |
| **Boss气旋** | 特定Boss技能产生的大型气旋（吞弹量x5） | 低频/事件 |

#### 5.1.6 玩家衍生战术（正向 × 负向 双向博弈）

**正向（玩家主动利用）**：

1. **延迟齐射**：故意把自己的子弹打进气旋，等释放时一次性高速强化输出 → 破解高血量精英怪
2. **延迟射击**：把气旋当成"弹药仓库"，先存弹、后选择最佳方向释放（方向继承=玩家说了算）
3. **加速暴击**：利用气旋的速度→伤害放大效应，把低伤害武器（如散弹）变成单发高伤
4. **风向裹挟联动**：气旋释放后，子弹仍受全局风带二次偏折，可形成"两段式弹道"

**负向（玩家需警惕）**：

5. **敌弹强化反噬**：气旋同样强化敌方子弹，**且按"方向继承"释放——会朝玩家最后射击方向飞**。若玩家最后射击方向恰好是自己的退路，敌弹被强化后顺着退路追杀玩家
6. **方向陷阱**：玩家在气旋附近的最后一次射击会决定释放方向，老练的敌人会等玩家射击后立刻贴近气旋，让强化后的敌弹与玩家"对冲"
7. **存满风险**：气旋容量打满后再射的弹直接消散——既不能塞太多，也要警惕敌人"灌满"气旋让玩家弹无法进入

> **核心博弈**：气旋是一个**双刃放大器**——它放大伤害也放大风险。玩家高手会把这个机制玩成"风险选择题"，新手可能把它当成单纯的输出装置。

## 6. 推荐 MVP 切片（3选1）

**MVP目标**：用一个5-10分钟的体验段落验证"超旋大气是否成立"——玩家能否感受到"风把游戏改变了"。

### 切片A · 风之走廊（最低成本，验证 M1+S1+G2）

- **场景**：单一房间，左侧是顺风走廊，右侧是逆风走廊，中间一面墙
- **机制覆盖**：G1风向场 + G2脉动 + M1风矢叠加 + S1风偏弹道
- **敌人**：3-5个 E1风骑兵
- **验证目标**：玩家在3分钟内是否能自然感受到"风的存在"且不别扭
- **资产成本**：⭐（最低，复用现有Room SO）
- **推荐优先级**：**P0**（先做）

### 切片B · 风眼挑战（验证奇景核心张力）

- **场景**：一片乱流带 + 中央风眼，风眼内悬浮目标物
- **机制覆盖**：G1+G2+G4+G5 + M1+M3+M5 + S1 + E1+E5
- **敌人**：风骑兵巡逻 + 迁徙群作为相位指示器
- **验证目标**：玩家是否能学会"在强相位躲、在弱相位冲"的节律
- **资产成本**：⭐⭐（中等，需新做风眼VFX）
- **推荐优先级**：P1

### 切片C · 风帆解谜（验证规则改写×关卡）

- **场景**：一座断风塔废墟，需激活L4断风器才能通过L1风门
- **机制覆盖**：G1+G2 + M1+M6 + S1 + L1+L4+L6 + E2
- **敌人**：风沙拟态在静风区显形
- **验证目标**：奇景是否能产生**新策略**而不只是新难度
- **资产成本**：⭐⭐⭐（最高，需新做机关交互）
- **推荐优先级**：P2

### 切片D · 气旋竞技场 ⭐（验证地面气旋族）

- **场景**：一个开阔房间，每隔10s在随机位置生成1-2个地面气旋
- **机制覆盖**：G1 + M1 + S1 + L8（含 L8-a/b/c/d + 聚焦齐射模式）+ E1
- **敌人**：4-6个 E1风骑兵
- **验证目标**：①子弹绕飞的视觉是否震撼 ②玩家是否能学会"延迟齐射"战术 ③气旋作为弹反装置是否好玩
- **资产成本**：⭐⭐（中等，主要是气旋VFX + 子弹轨道运动）
- **推荐优先级**：**P0/P1**（与切片A并列推荐，看你想先验证"风感"还是"局部装置玩法"）

| **L8** | 地面气旋（含 a/b/c/d 全套）| 本次设计亮点，视觉震撼+战术深度爆点 | ⭐⭐⭐⭐ 高 |
| L8 释放模式 | **方向继承**（已定）| L8 的灵魂 | ⭐ 极低 |
| L8-c | 气旋强化（速度→伤害）| 让 L8 不只是装置而是伤害放大器 | ⭐⭐ 低（命中已有"速度=伤害"规则）|
| **E1** | 风骑兵（症状型怪物）| 验证"怪物-奇景融合"是否成立 | ⭐⭐⭐ 中（顺风冲撞 AI + 弱相位逻辑） |

### 10.2 切片 D' 关键参数（已拍板）

| 参数 | 数值 | 说明 |
|---|---|---|
| **场景布局** | 中等开阔房间，左半区顺风带、右半区逆风带，中间区域生成气旋 | 同时验证 M1+S1（左右带）和 L8（中区） |
| **气旋生成频率** | 每 10s 在随机位置生成 1-2 个 | 让玩家有"等气旋来"的张力 |
| **气旋容量 N** | **15 发** | 比 20 略低，避免视觉过密 |
| **气旋吸吮期时长** | **4 秒** | 折中值（生成 0.5s + 吸吮 4s + 释放 0.3s） |
| **气旋加速倍率上限** | **×2.5** | 折中值，×3 太爆炸、×1.5 不够刺激 |
| **释放方向** | **方向继承**（朝玩家最后一次射击方向）| 唯一模式，其他变体不做 |
| **敌人配置** | 4-6 个 E1 风骑兵 | 单一敌人种类降低复杂度 |
| **风相位脉动周期** | 45-90s（仍待手感调参）| 进开发后再细调 |

### 10.3 验证目标（5 条）

切片 D' 跑出来后必须能回答以下5个问题，否则 MVP 不算成功：

1. **风感**：M1+S1 是否让玩家自然感受到"风的存在"且不别扭？
2. **节律**：G2+G3 的强弱相位脉动是否让玩家产生"等下一次风暴"的期待？
3. **气旋视觉**：L8-a/b 的子弹绕飞是否产生足够震撼的视觉？
4. **正向战术涌现**：玩家是否能自然学会"延迟齐射"战术？
5. **负向博弈成立**：敌弹反噬是否让玩家产生"开火即决策"的紧张感？

### 10.4 次轮扩展（验证成功后再做，6 个）

> 仅在 10.3 的 5 条验证目标至少命中 4 条时进入。

| ID | 机制 | 触发条件 |
|---|---|---|
| G4/G5 | 强相位/弱相位套件 | 验证"节律"成立后立即扩展 |
| M5 | 风眼真空 | 加入第一个反差区域 |
| S2 | 重弹绕飞 | 把"读风射击"扩展为"用风射击" |
| L1 | 风门 | 验证"奇景门控"（柱F）|
| E5 | 迁徙群（时钟指针型）| 让玩家可"预读相位" |
| L8 气旋类型扩展 | 触发气旋 + Boss气旋 | 把 L8 玩法做透 |

### 10.5 暂不落地（明确推迟）

- **全局机制**：G6 超级脉动、G7 静风事件、G8 五大风区
- **移动机制**：M2 风滑流、M3 风跳、M4 乱流眩晕、M6 风影锚点
- **射击机制**：S3-S7（激光、散弹梳化、风刃大招、声爆弹、风蚀）
- **敌人机制**：E2-E4、E6、E7
- **关卡机制**：L2-L7
- **L8 释放模式扩展**：径向爆发、聚焦齐射、风向裹挟
