# Project Ark 奇景候选池 (Wonder Candidate Pool)

> **文档目的**：为 Heart-Wonder（柱 A）和每颗星球的次级奇景储备"已通过审美筛选"的候选概念库。
> **定位**：原型素材库，非设计决策。最终选用哪个进入星球由 Planet Bible 决定。
> **上游文档**：`Ark_LevelDesignPillars.md`（柱 A · Heart-Wonder / 柱 B · Wonder Rhythm 的约束由此处承接）
> **版本**：v0.4 | 2026-05-10
> **状态**：🌱 初稿建立中 — 已收录 8 条单点奇景条目（§1）+ 2 条星球级候选（§2，OW 血统）；审美坐标 + 共性诊断 + 大师作品参照已入档

---

## 审美坐标（来自 2026-05-02 诊断）

### 审美支柱（候选条目必须至少命中其一）

- **X · 空间/时间几何反常** — 严肃硬科幻气质，关注几何/拓扑/时间的反直觉重构
- **Y · 场域波动可感知化** — 物理现象审美化，让不可见的数学/物理概念拥有视觉语言

### 2D 俯视角附加约束

- 地表型、无形场域、规则级、无情、持续，而非事件化/情节化
- **"星球运行方式本身就是奇观"**，而非"星球上有奇观"

---

## 0.5 用户审美的 7 维共性（首选 5 奇景反推）

> **基础数据**：用户 2026-05-02 选出的 5 个自评最佳奇景——螺旋下陷的漏斗、可见声音、两张重叠的地图、超旋大气、引力波驻波。
> **方法**：对这 5 个奇景做交集式维度分析，得出用户审美 DNA。

| # | 共性维度 | 描述 | 反面（不符合此品味的设计） |
|---|---------|------|--------------------------|
| 1 | **全星球弥漫式** | 是整颗星球的环境物理本身，而非定点式奇景 | "某座山上有个X"类的地标型奇景 |
| 2 | **无形的力量** | 都是场域 / 规则 / 波动，而非具体实体物件 | 巨大雕像、巨型生物、具象建筑群 |
| 3 | **无情的奇景** | 不响应玩家，体现"宇宙不在乎你在不在"的哲学 | 被玩家靠近就激活/攻击的反应型Boss式奇景 |
| 4 | **物理/数学概念直接可视化** | 对应真实物理概念（引力势阱、波传播、量子叠加等） | 纯幻想色彩、脱离真实物理参照的奇观 |
| 5 | **持续进行时** | 恒常背景压力，而非戏剧性节拍 | 事件化/爆发性/一次性演出 |
| 6 | **改变移动/感知规则** | 改变游戏元规则而非增加威胁数值 | 纯数值性的"更难的敌人" |
| 7 | **适合 2D 俯视角** | 所有奇景都天然适配用户的视角约束 | 依赖抬头看天/垂直纵深的视觉奇观 |

### 审美诊断的五字凝练

> **场域型 × 物理系 × 规则级 × 无情 × 持续**

这是所有候选条目必须通过的审美闸门——凡不命中这五个限定词的构想，归入"有创意但不对味"，不入候选池。

## 0.6 大师作品参照系

> 用户审美与下列作品/作者的"世界本身有问题"科幻美学家族强烈共鸣。
> 新候选条目在设计评估时，可问"它让人想起这些作品中的哪一件事物？"——答得出来，说明审美对位。

### 游戏 / 影视 / 文学类大师作品（5 件 + 扩展）

| 作品 | 介质 | 贡献的审美维度 | 对本项目的指导意义 |
|------|------|---------------|-------------------|
| **Outer Wilds** | 游戏 | "知识即进度"、活体动态宇宙 | Heart-Wonder + 三重揭示的直接原型 |
| **Annihilation（湮灭）** | 小说 / 电影 | Area X：规则本身被改写的世界 | "不可理解的场域"的氛围基准 |
| **Solaris（索拉里斯星）** | 小说 / 电影 | 整颗星球是一个不可沟通的存在 | "无情的奇景"美学的源头 |
| **Arrival（降临 / 你一生的故事）** | 小说 / 电影 | 时间观 / 语言的非线性 | 时间异常类奇景的参照 |
| **Dark Souls / Bloodborne** | 游戏 | 世界观通过环境自己发声 | 类魂压迫感的另一种表达路径 |

### 审美支柱对应的作者基准

| 支柱 | 代表人物 | 作品特征 |
|------|---------|---------|
| **X · 空间/时间几何反常** | **Greg Egan**（《Diaspora》《Schild's Ladder》《Orthogonal》三部曲） | 把高阶数学/物理变成可居住的世界 |
| **Y · 场域波动可感知化** | **3Blue1Brown**（Grant Sanderson，YouTube 数学可视化） | 让不可见的数学概念拥有视觉语言 |

### 使用方式（自查）

- 新候选条目应能"塞进"上述作品家族，哪怕只是一角
- 若一个构想**无法让人联想起任何上述作品**，多半是审美偏移，请重新审视
- 若一个构想**同时勾起 2 个以上作品的印象**，说明审美浓度高，值得重点孵化

| **用户评级** | ✅ 已认可 / 🤔 待进一步讨论 / ⏸️ 暂缓 |

## 1. 已认可条目（本轮收录）

### 1.1 配对守恒 (Paired Conservation)

| 栏位 | 内容 |
|------|------|
| **一句话定义** | 星球上每当产生一个事物，必然同步产生其"反物"，两者命运绑定。 |
| **所属类别** | E · 守恒律异常 |
| **审美支柱** | X · 空间/时间几何反常（偏向物理律层）|
| **柱 A 准则匹配** | 第一眼可见 ⭐⭐⭐（需设计显性视觉符号） / 持续搏动 ⭐⭐⭐⭐（凡玩家行为皆触发） / 与玩家张力 ⭐⭐⭐⭐⭐（每一次获取都伴随"另一端在哪里"的悬念） / 三重揭示潜力 ⭐⭐⭐⭐⭐（恐惧: 不明所以的反物出现 → 理解: 发现成对规律 → 驾驭: 主动配对博弈） / 辐射全星球 ⭐⭐⭐⭐⭐（物理律级，自动辐射） |
| **2D 俯视角适配度** | ⭐⭐⭐⭐⭐ — 不依赖视觉高度/天空，完全由地表行为可视化 |
| **用户评级** | ✅ 已认可（2026-05-04） |

**设计钩子（待展开）**：
- 战斗层：击杀一个敌人 → 远处同步"反生成"一个对偶体，必须找到并处理
- 探索层：拾取一个物品 → 某处出现"反物品"，二者必须同时激活才生效
- 叙事层："配对守恒"可以是星球古文明崩溃的原因——他们曾试图只取不还
- 与柱 E（星际回响）契合方向：三颗心脏可构成一组"三相守恒"命题的三种答卷

### 1.2 非均匀时间流 (Non-Uniform Time Flow)

| 栏位 | 内容 |
|------|------|
| **一句话定义** | 星球不同区域的时间流速不同，地表如同被时间河流的漩涡和急流覆盖。 |
| **所属类别** | B · 时间异常 |
| **审美支柱** | X · 空间/时间几何反常（时间几何的典型样本）|
| **柱 A 准则匹配** | 第一眼可见 ⭐⭐⭐⭐（植被/粒子/敌人动作速度差异可直接显化） / 持续搏动 ⭐⭐⭐⭐⭐（永不静止） / 与玩家张力 ⭐⭐⭐⭐⭐（好奇 + 紧迫 + 策略并存） / 三重揭示潜力 ⭐⭐⭐⭐⭐（恐惧: 诡异快慢 → 理解: 绘制时间地形 → 驾驭: 利用时间差规划路径与战术） / 辐射全星球 ⭐⭐⭐⭐⭐（时间是底层参量） |
| **2D 俯视角适配度** | ⭐⭐⭐⭐⭐ — 地表可视化天然适合（速度场、粒子漂移率、动画速率） |
| **用户评级** | ✅ 已认可（2026-05-04） |

**设计钩子（待展开）**：
- 战斗层：玩家在慢区攻速减半但也更易躲避；快区相反
- 探索层：存在"只有在快流区才生长/只有在慢流区才腐朽"的可解谜素材
- 导航层：时间流速场本身就是"地图信息"，替代传统地图标记
- 与柱 B（Wonder Rhythm）契合方向：全星球心跳在不同时流区产生不同周期的表达
- 注意事项：避免与其他星球的"时间型"主导子类型直接冲突；本条目可作为"时间型子类型"的一个实例表达

### 1.3 空间褶皱 (Space Folds)

| 栏位 | 内容 |
|------|------|
| **一句话定义** | 星球表面的空间存在可见的折叠/重叠区域，两点的直线距离与实际行走距离严重不符。 |
| **所属类别** | F · 拓扑结构异常 |
| **审美支柱** | X · 空间/时间几何反常（拓扑几何的典型样本）|
| **柱 A 准则匹配** | 第一眼可见 ⭐⭐⭐⭐⭐（褶皱接缝可设计为发光地貌） / 持续搏动 ⭐⭐⭐（需设计褶皱的呼吸/漂移） / 与玩家张力 ⭐⭐⭐⭐（迷失感 + 捷径诱惑） / 三重揭示潜力 ⭐⭐⭐⭐⭐（恐惧: 地图不闭合 → 理解: 褶皱规律 → 驾驭: 用褶皱走捷径/反打敌人） / 辐射全星球 ⭐⭐⭐⭐（拓扑级影响房间连接） |
| **2D 俯视角适配度** | ⭐⭐⭐⭐⭐ — 俯视角下拓扑异常最容易被眼睛读到（两张重叠的地图同源） |
| **用户评级** | ✅ 已认可（2026-05-04） |

**设计钩子（待展开）**：
- 探索层：同一张地图的两个房间实为同一物理空间的不同"层"
- 战斗层：敌人可从褶皱另一侧出现，打破传统 2D 俯视角的视野边界
- 与"两张重叠的地图"候选概念同源，可考虑二者合并或作为同一主题的不同实现
- 与柱 D（辐射物理学）契合方向：褶皱规则可辐射到小房间级别（小范围局部空间也是非欧）

### 1.4 螺旋下陷的漏斗 (Spiral Sinkhole Funnel)

| 栏位 | 内容 |
|------|------|
| **一句话定义** | 整颗星球的地表物质沿巨大螺旋缓慢下陷，永不见底，玩家站在一张正在流动的旋涡之上。 |
| **所属类别** | C · 场域压力（重力/几何复合） |
| **审美支柱** | X · 空间/时间几何反常 + Y · 场域波动可感知化（双支柱命中）|
| **柱 A 准则匹配** | 第一眼可见 ⭐⭐⭐⭐⭐（螺旋纹路是星球的视觉标识） / 持续搏动 ⭐⭐⭐⭐⭐（地表永远在缓慢旋转下沉） / 与玩家张力 ⭐⭐⭐⭐⭐（敬畏 + 缓慢死亡压力，不站稳就会被带走） / 三重揭示潜力 ⭐⭐⭐⭐⭐（恐惧: 被拖向深渊 → 理解: 螺旋速率场可读 → 驾驭: 借力下陷达到原本不可达的深层） / 辐射全星球 ⭐⭐⭐⭐⭐（所有房间都在流动基底上） |
| **2D 俯视角适配度** | ⭐⭐⭐⭐⭐ — 俯视角下螺旋纹路是最强的视觉识别符号 |
| **用户评级** | ✅ 已认可（2026-05-02，属"场域型×物理系×规则级×无情×持续"最高契合） |

**设计钩子（待展开）**：
- 基底层：整颗星球地表被"螺旋速率场"覆盖，远处物体会被持续拖走
- 探索层：越靠近漩涡中心，下陷速率越快，浅层/深层区域是同一张地图在不同时间的切片
- 战斗层：敌人和玩家都会被拖动，站位博弈从静态变成动态
- 与示巴星冰原兼容版本："漏斗冰原"—— 整片冰原在螺旋流动中缓慢重塑
- 与柱 B（Wonder Rhythm）契合方向：每一圈下陷对应一次"星球心跳"
- 可能的 Heart-Wonder 草案（示巴星备选）

### 1.5 可见声音 (Visible Sound)

| 栏位 | 内容 |
|------|------|
| **一句话定义** | 声波在这颗星球有可见形态，空气里持续涌动着肉眼可辨的驻波、涟漪与声流图案。 |
| **所属类别** | G · 共振同步 + D · 信息感知异常（复合） |
| **审美支柱** | Y · 场域波动可感知化（3Blue1Brown 级别可视化美学的直接体现）|
| **柱 A 准则匹配** | 第一眼可见 ⭐⭐⭐⭐⭐（声音图案就是星球的永恒风景） / 持续搏动 ⭐⭐⭐⭐⭐（声音永不静止） / 与玩家张力 ⭐⭐⭐⭐（好奇 + 信息过载感） / 三重揭示潜力 ⭐⭐⭐⭐⭐（恐惧: 无法理解的图案 → 理解: 声源与图案对应 → 驾驭: 用自身动作发声操纵图案） / 辐射全星球 ⭐⭐⭐⭐⭐（声学即物理律） |
| **2D 俯视角适配度** | ⭐⭐⭐⭐⭐ — 俯视角恰好能完整展示驻波图案（克拉德尼图形天然 2D） |
| **用户评级** | ✅ 已认可（2026-05-02，与示巴星既有"声学显形定律"高度兼容） |

**设计钩子（待展开）**：
- 感知层：玩家自身的移动/攻击/呼吸也会在空气中留下可见声纹
- 探索层：古文明遗址的隐藏信息编码在驻波图案里，需特定声源频率才能显形
- 战斗层：敌人通过声纹暴露位置；隐身 = 不发声
- 兼容性：与示巴星的"冰雕旋律碎片""音叉树""克拉德尼花"是同一底层物理律的不同表达
- 与柱 D（辐射物理学）契合方向：直接接管示巴星既有的声学机制宇宙
- 可能的 Heart-Wonder 草案（示巴星强备选，复用现有资产）

### 1.6 两张重叠的地图 (Two Overlapping Maps)

| 栏位 | 内容 |
|------|------|
| **一句话定义** | 星球同一地理坐标同时存在两个版本，玩家可在两张地图间切换/穿越，二者因果纠缠。 |
| **所属类别** | F · 拓扑结构异常 + B · 时间异常（复合）|
| **审美支柱** | X · 空间/时间几何反常 |
| **柱 A 准则匹配** | 第一眼可见 ⭐⭐⭐⭐（两版世界的视觉差异需要早期教学强调） / 持续搏动 ⭐⭐⭐⭐（切换时刻即搏动） / 与玩家张力 ⭐⭐⭐⭐⭐（一张地图的行为影响另一张，抉择密度极高） / 三重揭示潜力 ⭐⭐⭐⭐⭐（恐惧: 同一地点不同世界的诡异 → 理解: 两图因果映射 → 驾驭: 跨图规划路径与战术） / 辐射全星球 ⭐⭐⭐⭐⭐（是地图本身的结构规则）|
| **2D 俯视角适配度** | ⭐⭐⭐⭐⭐ — 俯视角 2D 下两张地图叠印是最天然的可视化语言 |
| **用户评级** | ✅ 已认可（2026-05-02） |

**设计钩子（待展开）**：
- 结构层：两张地图可能是"过去/现在"、"物质/幽灵"、"冰封前/冰封后"的二元
- 解谜层：一边开门，另一边通行；一边点火，另一边熄灭
- 战斗层：敌人只存在于其中一张地图，切换是躲避/伏击手段
- 与已收录 §1.3「空间褶皱」的关系：可视为空间褶皱的一个极端化特例（全局褶皱），或作为独立的"相位分裂"主题
- 与柱 C（三重揭示）契合方向：第三阶段允许玩家主动驾驭"相位切换"，类似 Titanfall 2《时光之间》关卡
- 可能的 Heart-Wonder 草案（星球 2 / 星球 3 备选，示巴星若选用则与空间褶皱择一）

### 1.7 超旋大气 (Hyper-Rotating Atmosphere)

| 栏位 | 内容 |
|------|------|
| **一句话定义** | 大气层以远超地表自转的速度高速旋转，形成永不停歇的环状气流压力带，横切整颗星球。 |

### 2.3 共生潮星 (Symbiotic Tide)

| 栏位 | 内容 |
|------|------|
| **一句话定义** | 整颗星球被两种活体物质（红藻/蓝藻）覆盖，昼夜交替时一方覆压一方退入地下，**同一地点白天与夜晚完全是两种地貌**。 |
| **OW 血统** | 灰烬双星（Hourglass Twins）的"此消彼长"机制 |
| **物理前提** | 红蓝两种活体表层互为宿敌；白天红强、夜晚蓝强；两者占据**同一套地形**但形态完全不同。 |
| **Heart-Wonder 地貌** | 整颗星球像一张**活的地图在呼吸**——白天是丛林、夜晚变成水晶洞窟，地形不变、表皮全换。 |
| **倒计时形式** | **日夜交替型**（建议星球内 1 天 ≈ 现实 15 分钟，比 OW 22 分钟更宽松，避免与星球 2 时间循环紧迫感复用）。白天能走的路径夜晚不能，反之亦然——形成**双层关卡**。 |
| **隐藏核心** | 星球赤道上有一条**永恒潮间带**——红蓝永远平衡的窄带，那里长着**唯一能穿越昼夜的植物**，作为核心指引。 |
| **单一情绪** | 节律 / 生命 / 不可抗争（与其他星球的"静"形成对照，是三星球唯一一颗"活"的）|
| **所属类别（§1 对位）** | C · 场域压力 + B · 时间异常 + 生态层（复合）|
| **审美支柱** | Y · 场域波动可感知化（活体潮汐的可视化）|
| **柱 A 准则匹配** | 第一眼可见 ⭐⭐⭐⭐⭐（红/蓝双色覆盖率反差极强） / 持续搏动 ⭐⭐⭐⭐⭐（永远在涨退） / 与玩家张力 ⭐⭐⭐⭐（节律压力，非致命） / 三重揭示潜力 ⭐⭐⭐⭐⭐（恐惧: 地形莫名消失 → 理解: 昼夜双层 → 驾驭: 卡时机切换路径） / 辐射全星球 ⭐⭐⭐⭐⭐ |
| **2D 俯视角适配度** | ⭐⭐⭐⭐⭐ — 俯视角下双色潮汐边界的推进/退却是教科书级视觉语言 |
| **审美闸门自检** | 场域型✅ × 物理系✅ × 规则级✅ × 无情✅（不响应玩家） × 持续✅ |
| **用户评级** | ✅ 已认可（2026-05-10） |

**设计钩子（待展开）**：
- 移动层：玩家必须在一日内完成两轮不同的探索；迟到/早到都会错过关键窗口
- 战斗层：双阵营敌人天然存在（红藻共生兽 vs 蓝藻共生兽），与双摇杆射击适配度高
- 探索层：赤道潮间带是永久可达区，是玩家在这颗星球的"家"基地
- 解谜层：某些谜题需要**同一地点白天放置物体、夜晚激活**（跨昼夜因果）
- 资产复用层：同一地形跑两套表皮，**美术资产 ROI 极高**
- 与柱 B（Wonder Rhythm）契合方向：日夜本身就是星球节律
- 与柱 H（单星球类型独立）契合方向：本星球类型 = "节律探险"，与其他星球区隔
- 三星球落位建议：作为**第二颗或第三颗星球**的候选（需与"星球 2 时间循环"差异化定位——本星球节律温和、有"家"、强调生命；时间循环紧迫、致命、强调宿命）

### 2.5 未完成星 (Unfinished)

| 栏位 | 内容 |
|------|------|
| **一句话定义** | 这颗星球还在**被建造**——某个宇宙级力量在塑造它，但进度只有 60%；玩家的探索行为本身会推进/锁定施工。 |
| **OW 血统** | 量子月（Quantum Moon）的"凝视即塑形" + 挪麦碎片（Nomai Probe Cannon）的"未完工感" |
| **物理前提** | 星球同时存在三种状态地表：**已完工区 / 施工中区 / 虚空未定义区**；空白区会在玩家**凝视它时**被随机"生成"出新地形。 |
| **Heart-Wonder 地貌** | 俯视角下**三种截然不同的视觉风格拼接在同一张地图上**——完整的山脉 / 半成型的低多边形网格 / 纯色空白区。视觉割裂本身就是奇观。 |
| **倒计时形式** | **玩家参与式倒计时**（不是时钟而是**进度条**）——玩家走过的空白区会被"确定"。**探索完整个星球，施工就结束了，奇景永远消失，星球变成一颗普通星球**。这是"不可挽回性"共性原则的星球级体现。 |
| **隐藏核心** | 一位**未完成的造物者**——他/它正在失败地创造这颗星球。玩家发现后要选择：**帮他完成**（星球凡俗化）/ **让他永远失败**（星球永远是奇观但造物者永堕）/ **取代他**（玩家成为造物者，柱 C 多结局体现）。 |
| **单一情绪** | 敬畏 / 造物感 / 元层面质疑（这颗星球**知道自己是被设计的**，是 Ark 唯一一颗反身性星球）|
| **所属类别（§1 对位）** | F · 拓扑结构异常 + D · 信息感知异常 + 元层（复合，元层为新维度）|
| **审美支柱** | X · 空间/时间几何反常（不完整的几何）+ 元层共鸣 |
| **柱 A 准则匹配** | 第一眼可见 ⭐⭐⭐⭐⭐（视觉风格割裂震撼） / 持续搏动 ⭐⭐⭐⭐（凝视即塑形，玩家行为驱动） / 与玩家张力 ⭐⭐⭐⭐⭐（探索即毁灭，存在主义压力） / 三重揭示潜力 ⭐⭐⭐⭐⭐（恐惧: 以为是 Bug → 理解: 这是设计 → 驾驭: 玩家就是塑造者本身） / 辐射全星球 ⭐⭐⭐⭐⭐ |
| **2D 俯视角适配度** | ⭐⭐⭐⭐⭐ — 美术资产可"合法地"留白（未定义区就是空白），**省资产但更震撼** |
| **审美闸门自检** | 场域型✅ × 物理系✅ × 规则级✅ × 无情✅（造物者不在乎玩家） × 持续✅ × 元层✅ |
| **用户评级** | ✅ 已认可（2026-05-10） |

**设计钩子（待展开）**：
- 玩家凝视塑形机制：玩家镜头/视野中心覆盖到的"虚空未定义区"会被随机抽取地形 prefab 生成；一旦生成不可撤销
- 探索悖论：每探索一寸土地就锁定一寸奇景；**玩家会主动停下探索来保留奇观**——逆传统玩家驱动力的设计
- 第三幕揭示：玩家发现造物者后，**整个星球被收回**（成为"游戏存档外的星球"），这是"超越屏幕边界"维度的体现
- 视觉资产策略：未定义区可用纯色 / Wireframe / 低多边形 / 噪声纹理，美术成本极低但**叙事浓度极高**
- 与柱 E（星际回响）契合方向：本星球可解释为"另两颗星球的造物者也是这个未完成造物者"——三星球同源
- 与柱 I（认知引力，待落地）契合方向：玩家追求"看见造物者"的认知引力极强
- 战略储备建议：作为 **Ark 最终星球 / DLC 终章** 候选（情绪强度高、不可挽回性、元层冲击）；不建议作为开局星球
- 风险点：元游戏机制有玩家两极化风险（部分玩家无法 buy in），需在前期留细节铺垫，避免 Inscryption KM 路径过门槛过高
