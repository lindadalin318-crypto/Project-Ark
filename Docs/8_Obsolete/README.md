# Obsolete — Project Ark

## 职责

`8_Obsolete/` 用于归档**已失效、已被替代或只保留历史价值**的文档与数据。

## 组织规则

归档区按**新主树职责镜像分区**组织：

- [`Plan/`](./Plan/)
- [`GameDesign/`](./GameDesign/)
- [`TechnicalDesign/`](./TechnicalDesign/)
- [`WorkflowsAndRules/`](./WorkflowsAndRules/)
- [`GameData/`](./GameData/)
- [`Diagnostics/`](./Diagnostics/)
- [`Reference/`](./Reference/)

## 使用规则

- 现役区只保留当前有效主稿
- 被替代的旧版主稿、旧快照、历史 checklist 和旧数据稿都应迁入这里
- 归档文件仍保持原职责语义，避免变成纯时间垃圾堆
