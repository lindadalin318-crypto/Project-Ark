# Technical Design — Project Ark

## 职责

`2_TechnicalDesign/` 只承载**技术设计真相源**。

这里回答：

- 系统运行时怎么成立
- 模块怎么拆
- authority 在哪里
- 关键对象与数据流如何协作

这里不放：

- workflow / authoring rules
- validator / audit 规则
- 项目推进计划
- 历史快照诊断
