# 🪐 示巴星 (Sheba) 功能需求清单 - 3

基于《示巴星关卡心流与节奏控制-2.csv》的5幕完整设计，本清单涵盖从0:00到310:00（约5小时）的全部功能需求。

---

## 1. 核心系统与机制

### 1.1 世界时钟系统 (World Clock / 潮汐系统)
- **渐强期 (Crescendo)**：
  - 环境高亮，敌人活跃，散热效率正常/加快
  - 特定门/电梯在渐强期开启
  - 敌人获得攻击增益
- **渐弱期 (Diminuendo)**：
  - 环境昏暗，敌人部分休眠/隐形
  - 玩家无法散热（热量只升不降）
  - 潜行窗口期，但无法使用需要散热的武器
- **世界时钟崩坏 (ACT 4)**：
  - 渐强期逐渐缩短（60s→45s→30s→15s→0s）
  - 渐弱期逐渐延长
  - 最终完全归零（永恒渐弱），所有依赖机制失效
  - UI显示归零并闪烁红光

### 1.2 热量管理系统 (Heat System)
- **基础机制**：
  - 开火增加热量，停止开火自动散热
  - 热量100%时过热惩罚：无法攻击/冲刺
- **环境热量交互**：
  - 静音力场区域：热量每秒+1%（无法散热）
  - 静音走廊（ACT4）：热量自动上升
- **完美闪避降热**：
  - 在敌人攻击窗口期冲刺穿过攻击
  - 瞬间降低15-20%热量
  - 视觉反馈：蒸汽涟漪+热量条下降+音效
- **双降热选择**：
  - 共鸣水晶（安全被动）：靠近即清零热量+2秒无敌
  - 完美闪避（危险主动）：高风险高回报

### 1.3 噪音累积槽 (Noise Meter) - ACT 4 新增
- **基础机制**：
  - 在孵化洞穴开火增加噪音值
  - 偶尔开一枪可容忍（+15%）
  - 连续开火快速飙升
- **渐进后果**：
  - 30%：周围幼体开始骚动（视觉警告闪烁）
  - 60%：区域局部发狂（2-3个茧破裂释放腐化者）
  - 100%：全面爆发（所有茧炸裂+大量敌人+道德代价）
- **涟漪特性**：
  - 涟漪（声波脉冲）不增加噪音
  - 鼓励用涟漪探路而非实弹

### 1.4 星图部件系统 (Star Chart)
- **部件类型**：
  - 星核（实弹/重炮/涟漪）
  - 棱镜（改装）
  - 光帆（冲刺）
  - 伴星（自动化）
- **获取节奏**：
  - 0:00 开局自带：星核实弹+重炮
  - 35:00 战斗奖励：棱镜分形·双生
  - 55:00 精英掉落：星核涟漪
  - 80:00 探索奖励：棱镜流变·反弹
  - 110:00 叙事赠予：棱镜晕染·霜冻
  - 130:00 Boss掉落：光帆共鸣
  - 160:00 探索奖励：伴星自动机炮
  - 210:00 精英掉落：棱镜流变·穿透
  - 230:00 叙事赠予：棱镜晕染·净化
  - 250:00 Boss掉落：光帆调律
  - 270:00 探索奖励：伴星护盾无人机
  - 290:00 Boss掉落：星图碎片示巴

## 2. 武器与星图系统

### 2.1 双武器系统
- 主武器（左键）：实弹机枪，高射速清群
- 副武器（右键）：重炮，高伤害对精英
- 星图UI拖拽教学（0:00-5:00）

### 2.2 涟漪系统 (声波弹) - 核心解谜工具
- **独立快捷键**：长按专用键发射，非武器槽位
- **四大功能**：
  - 解密：射击壁画/机关触发特定效果
  - 推物：推开物理碎片露出隐藏通道
  - 显形：射击隐形回音路/平台使其显形
  - 净化：净化黑水开辟通路
- **战斗应用**：
  - 穿透坚盾歌者盾牌
  - 暴露幽灵共鸣者实体
  - 暴露黑水之母核心

### 2.3 棱镜改装系统
- **分形棱镜·双生 (2001)**：+2颗子弹
- **流变棱镜·反弹 (2013)**：子弹墙壁反弹
- **晕染棱镜·霜冻 (2004)**：减速效果
- **流变棱镜·穿透 (2023)**：对群神器
- **晕染棱镜·净化 (2082)**：对黑水双倍伤害
- **熵增棱镜·回声 (2106)**：隐藏收集奖励

### 2.4 光帆系统
- **共鸣光帆**：冲刺释放和谐声波净化黑水
- **调律帆**：冲刺转化敌人攻击为和弦冲击波（反伤周围敌人）

### 2.5 伴星系统
- **自动机炮 (4001)**：第一个伴星，自动射击
- **护盾无人机 (4002)**：最终战前防御补充

## 3. NPC系统：缕拉 (Lyra)

### 3.1 视觉与交流
- **外观**：淡金色小型鸣钟者，琥珀色核心光芒，左臂黑色裂纹
- **颜色情绪表达**：
  - 琥珀=平静/信任
  - 赤红=愤怒/恐惧
  - 深蓝=悲伤
  - 翠绿=好奇/惊讶
  - 极亮金=决绝
- **频率翻译**：守夜人解读振动为文字（偶有"信号失真"效果）
- **旋律传达**：关键叙事时刻演奏旋律表达心意

### 3.2 共情触发点
- **触发点#1**：索尔雕像（ACT3 Z3a）- 讲述索尔故事
- **触发点#2**：追捕记忆（ACT3 Z3e）- 指挥家追捕闪回
- **触发点#3**：索尔遗言（ACT4 Z4c）- 大静默真相+半身黑化

### 3.3 可选增益系统
- ACT2 首次成功互动：歌唱型爬行者中立+护盾
- ACT3 守夜人Boss战(安抚)：安抚操作容错率提高
- ACT3 合唱考验：歌唱者合唱效率+50%
- ACT4 孵化洞穴：幼体不孵化腐化者
- ACT4-5 危机阶段：高频脉冲震退敌人
- ACT5 最终Boss：歌唱者增益提升

### 3.4 永久状态变化
- **大静默感染**：左臂裂纹逐渐向躯干扩散
- **半身黑化**：ACT4c后半边身体黑色纹路，核心光芒微弱
- **消逝预兆**：持续消耗能量维持生命

## 5. 物理与环境交互

### 5.1 声学显形定律
- **临时平台**：克拉德尼花射击后绽放3秒成立足点
- **隐形回音路**：涟漪射击显形3秒
- **隐形平台**：涟漪显形隐藏路径

### 5.2 声悬浮场
- 靠近巨大音叉树时重力减弱
- 跳跃高度和滞空时间增加

### 5.3 环境可射击物
- **音叉树**：射击产生共鸣波震晕敌人
- **管风琴管道**：射击产生不同音高冲击波
- **管壁共鸣区**：射击震晕范围内敌人
- **可击碎玻璃花窗**：改变光照/地形

### 5.4 黑水系统
- **接触伤害**：即死或高伤
- **涟漪净化**：退缩露出通道
- **声悬浮对抗**：站上晶体平台可避开
