## Ship 特效玩家感知参考（现役版）

### 文档角色

这份文档只回答一个问题：

**玩家在当前 `SampleScene` 里，主观上实际能感受到哪些 `Ship` 视觉特效。**

它是**玩家感知视角参考**，不是命名、路径、owner 或迁移策略的权威文档。

当前现役判断基于以下来源交叉核对：

- `ShipView`
- `BoostTrailView`
- `ShipEngineVFX`
- `ShipVisualJuice`
- `DashAfterImageSpawner`
- `ShipVFX_CanonicalSpec.md`
- `ShipVFX_AssetRegistry.md`

这次版本特意做了删减，以下内容**不再**放进本表：

- dormant / legacy / 参考资源
- prefab 节点物理名、owner、迁移批次
- Editor 工具接线细节
- “修复了一个不该出现的问题”这类工程说明

## 一、常驻读感：玩家平时一直能看到的

| 玩家感知 | 现役效果 | 玩家实际看到 |
| - | - | - |
| **船是“活的”** | **主引擎尾焰** | 怠速时轻喷，移动越快尾焰越明显 |
| **转向有重量感** | **船体倾斜 Tilt** | 飞船横向移动时会轻微侧倾，不是死板平移 |
| **加速/减速更有弹性** | **普通移动 Squash / Stretch** | 普通加速会轻微拉伸，减速会轻微压缩，让速度变化更有肉感 |

---

## 二、Dash 爆发：玩家按下 Dash 时立刻感受到的

| 玩家感知 | 现役效果 | 玩家实际看到 |
| - | - | - |
| **进入闪避窗口** | **无敌帧闪烁** | 船体实色层会快速闪烁，强调“现在正在闪避” |
| **原地留下一张残像** | **静态 Ghost** | Dash 起点会留下一张短暂淡出的残像 |
| **高速划过空间** | **连续 After-Images** | Dash 轨迹上会连续刷出多张残影，形成拖影感 |
| **推进器瞬间拉满** | **引擎 Dash 档增强** | 尾焰会立刻更猛、更亮，强化冲刺爆发感 |

---

## 三、Boost 起手：玩家“进入 Boost 那一刹那”感受到的

| 玩家感知 | 现役效果 | 玩家实际看到 |
| - | - | - |
| **Boost 点火成功** | **Halo + Bloom 爆闪** | 船身周围会闪出一圈局部光环，同时整体亮度短促抬升一下 |
| **推进器被点燃** | **FlameCore + EmberSparks Burst** | 尾喷根部会先炸出一口核心火焰和一小波火星 |
| **机体进入高能态** | **液态层亮起并切到 Boost Sprite** | 船体液态层明显变亮，观感从普通态切到 Boost 态 |
| **尾喷先弹一下** | **尾喷 Entry Pulse** | 船尾喷口会先有一次明显脉冲，再进入持续输出 |

---

## 四、Boost 持续：玩家进入高速巡航态后持续看到的

| 玩家感知 | 现役效果 | 玩家实际看到 |
| - | - | - |
| **有一条稳定的高速尾迹** | **MainTrail 主拖尾** | 后方持续拖出主尾迹，形成最主要的高速读感 |
| **推进器两侧持续喷火** | **双侧 FlameTrail** | 两侧火焰稳定输出，强化推进器全开状态 |
| **尾迹还有余烬层次** | **EmberTrail** | 主拖尾之外还有更散、更轻的余烬轨迹 |
| **机体保持充能状态** | **能量层强度维持** | 机体上的能量层持续维持高亮，不会只在起手闪一下 |
| **尾部维持高功率呼吸** | **引擎 Boost 档 + 尾喷循环 Pulse** | 引擎尾焰锁在高档，船尾喷口持续有节奏地脉冲 |

---

## 五、Boost 收尾：玩家松开或结束 Boost 时感受到的

| 玩家感知 | 现役效果 | 玩家实际看到 |
| - | - | - |
| **高速尾迹不是硬切断** | **拖尾 / 火焰 / 能量层平滑收尾** | 主尾迹会自然拉尾，火焰和能量层也会一起慢慢退下去 |
| **机体退回普通态** | **液态层亮度回落 + Sprite 切回 Normal** | 船体从高能态逐渐恢复到平时的视觉状态 |
| **尾喷回到常规输出** | **尾喷 Restore + 引擎 Boost 档解除** | 船尾脉冲结束，引擎强度重新交回普通速度逻辑接管 |

---

## 六、一句话总结

从玩家主观体验看，当前 `Ship` 的现役读感可以概括为：

- **平时**：引擎尾焰 + 船体轻微倾斜 + 速度形变
- **Dash**：闪烁 + 原地残像 + 连续拖影 + 尾焰强化
- **Boost 起手**：光环 / bloom 爆闪 + 点火 burst + 机体亮起 + 尾喷脉冲
- **Boost 持续**：主拖尾 + 双侧火焰 + 余烬 + 能量层 + 高功率尾喷呼吸
- **Boost 结束**：所有高能态效果平滑收尾，而不是瞬间硬切
