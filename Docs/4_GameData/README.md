# Game Data — Project Ark

## 职责

`4_GameData/` 存放游戏中的结构化数据与特殊格式数据，包括：

- `.csv`
- `.json`
- 其他结构化数据文件
- 数据字典、schema、导入规则（后续按需补充）

## 组织规则

- 按**内容主题**分目录，而不是按文件格式分目录
- 世界 / 地区专属数据，优先按世界归类
- 通用系统数据，再按系统归类

## 当前目录

- [`Sheba/`](./Sheba/): 示巴星专属数据包
- [`Enemy/`](./Enemy/): 通用敌人与敌人规划数据
- [`StarChart/`](./StarChart/): 星图系统数据
- [`Level/`](./Level/): 通用关卡结构化数据预留目录
