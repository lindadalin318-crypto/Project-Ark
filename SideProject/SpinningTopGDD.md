## 1. 游戏概述

### 1.1 一句话定位

> **一款以战斗陀螺为核心古典原型，融合杀戮尖塔式爬塔结构与肉鸽 Build-Crafting 深度的策略动作游戏。**

### 1.2 参考游戏锚定

| 参考游戏 | 借鉴要素 |
|----------|---------|
| **Slay the Spire** | 爬塔结构、准备回合情报侦察、节点地图、经济决策 |
| **Ball x Pit** | 物理碰撞核心、零件进化（Fusion）、Survivor-like 波次压力、满屏物理视觉奇观 |
| **CloverPit** | 老虎机式随机性、赌博张力、恐怖氛围、道具协同爆炸 |
| **Beyblade（战斗陀螺玩具）** | 古典原型、四部件组装、三流派石头剪刀布、发射操作 |
| **台球游戏** | 力道卡 Timing 机制、角度瞄准 |

### 1.3 核心体验目标

1. **"我的 Build 无敌了"** — 零件组合产生的协同爆炸感
2. **"我要再来一把"** — 每局 15-20 分钟的短局时长 + 肉鸽随机性
3. **"我要看看下一层有什么"** — 爬塔结构的探索欲驱动
4. **"这一发太爽了"** — 发射 Timing + 物理碰撞的即时反馈

### 1.4 目标平台与受众

- **平台**：PC（Steam）优先，后续考虑移动端
- **受众**：肉鸽爱好者、战斗陀螺童年玩家、策略游戏玩家
- **单局时长**：15-20 分钟（一次完整 Run）
- **总游戏时长**：首通约 5-8 小时，深度玩家 50+ 小时

## 2. 核心设计哲学

### 2.1 三大设计支柱

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   🧠 策略深度          🎯 操作手感          🎲 随机惊喜   │
│                                                         │
│  情报侦察 + 零件       发射角度 + 力道       零件池随机 +  │
│  组合决策 + 经济       Timing + 物理碰撞     敌人配置随机  │
│  管理 = 每局不同       = 即时反馈爽感        = 每局故事    │
│  的 Build 路线                                           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 2.2 核心设计原则

1. **情报不完整** — 准备回合只能看到敌人的部分信息，决策永远在不确定性中进行
2. **物理是叙事** — 陀螺的碰撞、弹飞、停转都是有意义的战斗叙事，不是纯随机
3. **零件是角色** — 每个零件有独特的"个性"，玩家对自己的 Build 产生情感依附
4. **短局高密度** — 每一个决策都有重量，没有废局
5. **先 Work 再 Right** — 物理手感优先于功能完整性

## 3. 游戏循环结构

### 3.1 宏观循环（一次完整 Run）

```
开始 Run
    │
    ▼
┌─────────────────────────────────────────────────────────┐
│                    爬塔节点地图                           │
│                                                         │
│  Floor 1  ──▶  Floor 2  ──▶  Floor 3  ──▶  Floor 4     │
│  (3 节点)      (3 节点)      (3 节点)      (Boss)        │
│                                                         │
│  每个节点 = 一次 [准备回合] + [战斗回合]                  │
└─────────────────────────────────────────────────────────┘
    │
    ▼
击败 Final Boss → Run 结束 → 获得 Meta 奖励
```

### 3.2 微观循环（单个节点）

```
[情报侦察]
    │  系统随机展示敌人的 2-4 条信息
    │  （陀螺类型？部件？数量？特殊能力？）
    ▼
[准备回合 - Prep Phase]
    │  查看当前零件库存
    │  前往商店购买/升级零件
    │  决定本场使用的零件配置
    │  （第一节点：免费选择底座类型）
    ▼
[发射准备]
    │  选择发射角度（360° 瞄准）
    │  卡 Timing 决定力道大小
    ▼
[战斗回合 - Battle Phase]
    │  陀螺在竞技场中物理运行
    │  RPM 条作为生命值
    │  碰撞产生伤害和特效
    │  胜利条件：敌方 RPM 归零 / 场外 / 爆裂
    ▼
[战斗结算]
    │  获得金币 + 零件奖励（3 选 1）
    │  特殊节点：商店 / 事件 / 精英战
    ▼
[进入下一节点]
```

### 3.3 节点类型

| 节点类型 | 图标 | 频率 | 说明 |
|----------|------|------|------|
| **普通战斗** | ⚔️ | 60% | 标准准备+战斗流程 |
| **精英战斗** | 💀 | 15% | 敌人更强，奖励更丰厚 |
| **商店** | 🏪 | 10% | 纯购物，无战斗，可出售零件 |
| **神秘事件** | ❓ | 10% | 随机事件（风险/收益选择） |
| **Boss** | 👑 | 5% | 每层最终关卡，特殊机制 |

## 4. 准备回合（Prep Phase）

### 4.1 情报侦察系统

战斗开始前，系统从敌人的完整信息中**随机抽取 2-4 条**展示给玩家：

**敌人完整信息池（玩家不一定全看到）：**

| 信息类别 | 示例 | 策略价值 |
|----------|------|---------|
| **陀螺数量** | "本场有 2 个敌方陀螺" | 决定是否需要 AOE 零件 |
| **流派类型** | "敌方为攻击型" | 决定克制策略（防御型克攻击型） |
| **攻击环类型** | "敌方装备了火焰攻击环" | 决定是否需要抗性零件 |
| **配重盘信息** | "敌方使用重型配重盘" | 决定是否需要穿透零件 |
| **底座类型** | "敌方使用平面轴尖（移动型）" | 决定场地控制策略 |
| **特殊能力** | "敌方陀螺在低 RPM 时会爆发" | 决定是否快速击杀 |
| **竞技场信息** | "本场竞技场有磁力中心" | 决定发射角度策略 |

**设计意图**：信息不完整 = 决策永远有风险。玩家需要根据已知信息推断未知，形成"侦探式"的策略乐趣。

### 4.2 零件配置界面

准备回合玩家可以：

1. **查看当前零件库存** — 所有已拥有的零件
2. **装配陀螺** — 从库存中选择零件装配（每个槽位只能装一个）
3. **前往商店** — 花费金币购买新零件（消耗准备时间，但准备回合无时间限制）
4. **出售零件** — 将不需要的零件卖掉换取金币

### 4.3 第一节点特殊规则：底座选择

游戏开始进入第一个节点的准备回合时，玩家**免费从 3 种底座中选择 1 种**作为起始底座：

| 底座选项 | 类型 | 特性 | 推荐风格 |
|----------|------|------|---------|
| **尖锐轴尖（Sharp Tip）** | 防御型 | 原地高速旋转，RPM 衰减极慢，但移动范围小 | 稳健防守，等待对手自行衰减 |
| **平面轴尖（Flat Tip）** | 攻击型 | 大范围移动，碰撞力强，但 RPM 衰减较快 | 主动进攻，快速击倒对手 |
| **橡胶轴尖（Rubber Tip）** | 持久型 | 抓地力强，被撞后不易弹飞，RPM 衰减中等 | 耐久战，适合长期消耗 |

> **设计意图**：底座选择决定了整个 Run 的核心策略方向，类似杀戮尖塔的职业选择。底座是最贵的零件，不能在商店轻易更换（需要高额金币），所以这个初始选择非常重要。

## 5. 战斗回合（Battle Phase）

### 5.1 竞技场物理环境

**默认竞技场（Standard Arena）：**

- 圆形凹盘，直径约 60cm（游戏内比例）
- 盘面向中心有约 **15° 的倾斜角**，使陀螺自然向中心汇聚
- 盘面材质：光滑塑料，摩擦系数低
- 边缘有挡板，防止陀螺轻易飞出（但被强力撞击仍可飞出 = 场外胜）
- 物理模拟目标：**尽可能真实的陀螺物理**，包括进动（Precession）、章动（Nutation）、角动量守恒

**物理模拟重点：**

| 物理现象 | 游戏表现 | 策略意义 |
|----------|---------|---------|
| **进动（Precession）** | 陀螺轴心画圆，低 RPM 时更明显 | 低 RPM 陀螺轨迹不稳定，难以预测 |
| **碰撞反弹** | 碰撞角度决定弹射方向 | 发射角度影响碰撞后的轨迹 |
| **质量差异** | 重陀螺撞轻陀螺 = 轻陀螺被弹飞更远 | 配重盘选择影响碰撞结果 |
| **旋转方向** | 同向碰撞 = 摩擦减速；反向碰撞 = 剧烈弹飞 | 旋转核心的左/右旋选择有战术意义 |
| **倾斜角** | 盘面倾斜使陀螺向中心漂移 | 防御型陀螺天然占据中心优势 |

### 5.2 发射系统

#### 5.2.1 角度选择

- 玩家在发射前可以**360° 自由选择发射角度**
- 界面显示竞技场俯视图，玩家拖动瞄准线选择入射方向
- 可以看到敌方陀螺的大致位置（但敌方陀螺在发射前已经在运动）
- **策略维度**：
  - 直线冲向敌人 = 高碰撞力，但可预测
  - 切线入射 = 碰撞后产生旋转效果，难以预测
  - 边缘入射 = 利用盘面倾斜产生弧线轨迹

#### 5.2.2 力道系统（Power Gauge）

力道系统是战斗回合唯一的**主动操作**，借鉴台球游戏的 Timing 机制：

```
力道表示意图：

  100% ████████████████████████ MAX
   80% ████████████████████
   60% ████████████████
   40% ████████████
   20% ████████
    0% ████

  ↑↓ 力道条持续上下涨幅（正弦波形）
  玩家按下确认键的瞬间 = 本次发射力道
```

**力道影响：**

| 力道区间 | 效果 |
|----------|------|
| **90-100%（完美）** | 初始 RPM 最高，碰撞力最强，有金色特效 |
| **70-89%（良好）** | 正常发射，无特殊效果 |
| **40-69%（普通）** | 初始 RPM 略低，碰撞力减弱 |
| **0-39%（失误）** | 初始 RPM 很低，陀螺可能提前停转 |

**力道条速度**随游戏进程加快（精英战 > 普通战），增加操作压力。

**特殊零件可以影响力道系统**（见零件系统章节）。

### 5.3 战斗进行中

发射后，玩家**不能直接操控陀螺**，战斗自动进行。但玩家拥有有限的**干预能力**：

| 干预类型 | 冷却 | 效果 | 解锁条件 |
|----------|------|------|---------|
| **推力（Boost）** | 每场 1 次 | 给己方陀螺一次额外推力，改变运动方向 | 默认解锁 |
| **制动（Brake）** | 每场 1 次 | 短暂降低己方陀螺移速，躲避危险碰撞 | 特定底座解锁 |
| **激活（Activate）** | 每场 1 次 | 触发装备零件的特殊主动技能 | 特定零件解锁 |

**设计意图**：玩家不是"操控者"而是"指挥者"——大部分时间在观察战局，在关键时刻做出干预决策。这和杀戮尖塔的"出牌"类似，干预次数有限，每次都要想清楚。

### 5.4 胜负判定

| 胜利类型 | 条件 | 奖励加成 |
|----------|------|---------|
| **存活胜（Outspin）** | 敌方 RPM 归零先停转 | 标准奖励 |
| **场外胜（Ring Out）** | 敌方陀螺被撞出竞技场 | +20% 金币 |
| **爆裂胜（Burst）** | 敌方陀螺被撞散架（部件分离） | +50% 金币 + 额外零件奖励 |
| **失败** | 己方 RPM 归零 / 场外 / 爆裂 | 失去 HP（见爬塔结构） |

## 6. 零件系统（Parts System）

### 6.1 零件槽位总览

每个陀螺由 **4 个槽位**组成，每个槽位只能装一个零件：

```
        ┌─────────────────┐
        │   攻击环 (AR)    │  ← 顶层，决定碰撞攻击效果
        ├─────────────────┤
        │   配重盘 (WD)    │  ← 中层，决定物理属性
        ├─────────────────┤
        │   旋转核心 (SG)  │  ← 内核，决定旋转方向和特殊机制
        ├─────────────────┤
        │   底座/轴尖 (BB) │  ← 底部，决定运动模式（最贵，最重要）
        └─────────────────┘
```

### 6.2 攻击环（Attack Ring / AR）

攻击环决定陀螺碰撞时的**攻击效果和形状**。

#### 基础攻击环（无元素）

| 名称 | 形状特征 | 效果 | 价格 |
|------|---------|------|------|
| **尖刺环（Spike Ring）** | 多个尖刺突出 | 碰撞伤害 +30%，但被撞时自身也受到反弹伤害 | 80G |
| **圆滑环（Smooth Ring）** | 圆润无棱角 | 被撞时 RPM 损失 -20%（防御性） | 60G |
| **锯齿环（Saw Ring）** | 细密锯齿 | 碰撞时持续造成小量 RPM 伤害（类似出血） | 70G |
| **宽翼环（Wide Wing）** | 宽大翼展 | 碰撞范围 +40%，但碰撞力度 -10% | 75G |

#### 元素攻击环

| 名称 | 元素 | 碰撞效果 | 价格 |
|------|------|---------|------|
| **火焰环（Flame Ring）** | 🔥 火 | 碰撞后在接触点留下燃烧区域，持续造成 RPM 伤害 | 120G |
| **冰霜环（Frost Ring）** | ❄️ 冰 | 碰撞后使敌方陀螺移速降低 30%，持续 3 秒 | 120G |
| **雷击环（Thunder Ring）** | ⚡ 电 | 碰撞后产生连锁闪电，若场内有其他陀螺则跳跃伤害 | 130G |
| **毒素环（Venom Ring）** | ☠️ 毒 | 碰撞后使敌方陀螺 RPM 持续缓慢衰减（叠加层数） | 110G |
| **磁力环（Magnet Ring）** | 🧲 磁 | 碰撞后短暂吸引敌方陀螺，使其难以逃脱 | 140G |

#### 高阶融合攻击环（Fusion AR）

通过特定零件组合或商店购买解锁：

| 名称 | 融合来源 | 效果 |
|------|---------|------|
| **蒸汽环（Steam Ring）** | 火焰 + 冰霜 | 碰撞产生大范围蒸汽云，遮蔽视野 + 持续 RPM 伤害 |
| **辐射环（Radiation Ring）** | 雷击 + 毒素 | 碰撞后敌方陀螺变成"毒电源"，接触其他陀螺时传播伤害 |
| **极光环（Aurora Ring）** | 冰霜 + 电 | 碰撞后产生冰电爆炸，大范围 RPM 伤害 + 减速 |

### 6.3 配重盘（Weight Disk / WD）

配重盘决定陀螺的**物理属性**，影响碰撞结果和 RPM 衰减速度。

| 名称 | 重量 | 效果 | 价格 |
|------|------|------|------|
| **重型盘（Heavy Disk）** | 极重 | 碰撞力 +40%，被撞时不易弹飞，但移速 -20% | 90G |
| **轻型盘（Light Disk）** | 极轻 | 移速 +30%，RPM 衰减 -10%，但被撞时弹飞距离 +50% | 70G |
| **均衡盘（Balance Disk）** | 中等 | 无特殊加成，但 RPM 衰减速度最慢（持久型首选） | 80G |
| **偏心盘（Eccentric Disk）** | 中等 | 运动轨迹不规则，难以预测，碰撞角度随机 ±15° | 85G |
| **磁化盘（Magnetic Disk）** | 重 | 被磁力环吸引时免疫，同时对磁力环敌人有反制效果 | 100G |

### 6.4 旋转核心（Spin Gear / SG）

旋转核心决定**旋转方向**和**特殊被动机制**。

#### 旋转方向

| 类型 | 效果 |
|------|------|
| **右旋（Right Spin）** | 标准旋转方向 |
| **左旋（Left Spin）** | 与右旋陀螺碰撞时产生剧烈弹飞效果（反向碰撞）；与左旋陀螺碰撞时产生摩擦减速（同向碰撞） |

> **策略意义**：如果情报显示敌方是右旋，选择左旋核心可以在碰撞时产生更大的弹飞效果（场外胜概率提升）。

#### 特殊旋转核心

| 名称 | 效果 | 价格 |
|------|------|------|
| **标准核心（Standard Gear）** | 无特殊效果，左/右旋可选 | 50G |
| **加速核心（Turbo Gear）** | 每次碰撞后短暂提升自身 RPM（动能反馈） | 100G |
| **护盾核心（Shield Gear）** | 每场战斗开始时有一次免伤（吸收一次碰撞伤害） | 110G |
| **爆发核心（Burst Gear）** | 当己方 RPM 低于 30% 时，碰撞伤害翻倍（背水一战） | 120G |
| **共振核心（Resonance Gear）** | 与同元素攻击环配合时，元素效果持续时间 +50% | 130G |

### 6.5 底座/轴尖（Blade Base / BB）

底座是**最重要、最贵**的零件，决定陀螺的运动模式和核心战术风格。

> **规则**：一个陀螺只能有一个底座。游戏开始时免费选择，之后更换需要高额金币（150G+）。

| **橡胶轴尖（Rubber Tip）** | 橡胶底面 | 中等移动范围，抓地力强 | 被撞后弹飞距离 -40%（不易场外）；RPM 衰减中等 | 持久型 / 防御型 |

| 名称 | 价格 | 特性 |

| **磁力底座（Magnet Base）** | 220G | 特殊：在盘面上产生磁力轨道，陀螺沿固定轨迹运动（可预测但难以被撞飞） |

## 7. RPM 战斗机制

### 7.1 RPM 条设计

RPM（Revolutions Per Minute，每分钟转速）是陀螺的"生命值"，但比普通生命值更有叙事感：

```
RPM 条视觉示意：

  ████████████████████████████████  100% — 高速旋转，稳定
  ████████████████████████          75%  — 正常旋转
  ████████████████                  50%  — 开始出现轻微晃动
  ████████                          25%  — 明显晃动，进动加剧
  ████                              10%  — 剧烈摇摆，即将停转
  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░   0%  — 停转，战斗失败
```

**RPM 与视觉表现挂钩**：
- 100-75%：陀螺旋转稳定，轨迹流畅
- 75-50%：陀螺开始轻微晃动，旋转音效音调略降
- 50-25%：明显晃动，进动圆圈变大，轨迹不稳定
- 25-0%：剧烈摇摆，随时可能停转，音效变得沉重

### 7.2 RPM 变化规则

#### RPM 自然衰减

- 所有陀螺的 RPM 都在**持续自然衰减**
- 衰减速度受**底座类型**影响（尖锐轴尖衰减最慢，平面轴尖衰减最快）
- 衰减速度受**配重盘**影响（均衡盘衰减最慢）

#### RPM 碰撞伤害

碰撞时，双方陀螺都会损失 RPM，但损失量不同：

```
碰撞伤害计算（简化公式）：

攻击方 RPM 损失 = 基础碰撞力 × 攻击方质量系数 × 0.3（攻击方损失较少）
防御方 RPM 损失 = 基础碰撞力 × 攻击方质量系数 × 1.0（防御方损失较多）

基础碰撞力 = 攻击方移速 × 攻击方质量 × 攻击环加成
```

**关键设计**：碰撞对双方都有伤害，但攻击方损失更少。这意味着**主动进攻是有利的**，但也不能无脑冲撞——每次碰撞都有代价。

#### RPM 恢复机制

| 恢复来源 | 恢复量 | 条件 |
|---------|--------|------|
| **主动碰撞（平面轴尖）** | +5% RPM | 每次成功碰撞敌方 |
| **加速核心** | +8% RPM | 每次碰撞后触发 |
| **盘心区域（尖锐轴尖）** | +2% RPM/秒 | 在竞技场中心区域时 |
| **特殊零件效果** | 各异 | 见零件描述 |

**设计意图**：RPM 恢复机制使得"你必须保持攻击性"——停下来就死，碰撞才是生存。这和 Ball x Pit 的设计哲学一致。

### 7.3 RPM 阶段效果

| RPM 阶段 | 效果 |
|---------|------|
| **满血（75-100%）** | 无特殊效果 |
| **受伤（50-75%）** | 无特殊效果 |
| **危险（25-50%）** | 碰撞后弹飞距离 +20%（更容易被撞出场外） |
| **濒死（0-25%）** | 碰撞后弹飞距离 +50%；爆裂概率 +30% |

## 8. 敌人系统（Enemy System）

### 8.1 敌人三流派（石头剪刀布）

继承战斗陀螺的经典三流派设计，形成天然的克制关系：

```
    攻击型
   ↗       ↘
持久型 ← 防御型

攻击克持久（直接撞停）
防御克攻击（扛住碰撞，攻击型反弹自损）
持久克防御（防御型虽扛住但耗能快，持久型拖到最后）
```

### 8.2 基础敌人档案

#### 攻击型敌人

| 名称 | 特征 | 行为模式 | 弱点 |
|------|------|---------|------|
| **冲锋者（Rusher）** | 平面轴尖，尖刺攻击环 | 高速直线冲向玩家，碰撞力强 | 防御型底座可以扛住并反弹 |
| **旋风者（Cyclone）** | 平面轴尖，宽翼攻击环 | 大范围扫荡，碰撞范围广但单次伤害低 | 集中在一点的高伤害碰撞 |
| **爆裂者（Buster）** | 特殊底座，爆裂攻击环 | 低 RPM 时触发自爆，对周围造成大量伤害 | 快速击杀，不让其进入低 RPM 状态 |

#### 防御型敌人

| 名称 | 特征 | 行为模式 | 弱点 |
|------|------|---------|------|
| **堡垒（Fortress）** | 重型配重盘，圆滑攻击环 | 扎根中心不动，碰撞时反弹攻击方 | 元素伤害（绕过物理防御）|
| **镜盾（Mirror Shield）** | 特殊攻击环 | 将碰撞伤害的 50% 反弹给攻击方 | 持久消耗，不要正面碰撞 |
| **磁核（Magnet Core）** | 磁力底座 | 在固定轨道运动，吸引玩家陀螺靠近 | 磁化盘配重可以免疫吸引 |

#### 持久型敌人

| 名称 | 特征 | 行为模式 | 弱点 |
|------|------|---------|------|
| **耐久者（Endurer）** | 尖锐轴尖，均衡配重 | 原地旋转，RPM 衰减极慢，等待玩家先停 | 主动碰撞，不给它拖时间 |
| **毒雾者（Miasma）** | 橡胶轴尖，毒素攻击环 | 缓慢移动，留下毒素轨迹，场地越来越危险 | 快速击杀，避免在毒素区域停留 |
| **吸血者（Leech）** | 特殊旋转核心 | 每次被碰撞时吸取攻击方的 RPM | 持久消耗，避免频繁碰撞 |

### 8.3 精英敌人

精英敌人是强化版的基础敌人，拥有**额外的特殊能力**：

| 名称 | 基础类型 | 特殊能力 |
|------|---------|---------|
| **烈焰冲锋者（Inferno Rusher）** | 攻击型 | 碰撞后留下大范围火焰区域 |
| **冰霜堡垒（Frost Fortress）** | 防御型 | 被碰撞时释放冰霜爆炸，减速攻击方 |
| **永恒耐久者（Eternal Endurer）** | 持久型 | RPM 低于 20% 时自动恢复至 40% （一次性） |

### 8.4 Boss 设计

每层的 Boss 拥有**三阶段机制**，每个阶段切换不同的流派行为：

**示例 Boss：「混沌冠军（Chaos Champion）」**

| 阶段 | RPM 范围 | 行为模式 | 玩家应对 |
|------|---------|---------|---------|
| **第一阶段** | 100-60% | 攻击型：高速冲撞，主动进攻 | 防御或闪避，等待反击机会 |
| **第二阶段** | 60-30% | 防御型：扎根中心，反弹攻击 | 改用元素伤害，避免正面碰撞 |
| **第三阶段** | 30-0% | 持久型：释放毒素场地，拖延时间 | 快速进攻，不给它拖延 |

## 9. 爬塔结构（Tower Structure）

### 9.1 塔层设计

一次完整的 Run 包含 **4 层（Floor）**，每层有 **3-4 个节点**：

```
节点地图示意（类杀戮尖塔）：

Floor 4:              [BOSS]
                     /
Floor 3:    [精英] - [商店] - [普通]
           /                  \
Floor 2: [普通] - [事件] - [精英]
           \                  /
Floor 1:    [普通] - [普通] - [商店]
                     |
                  [起点]
```

### 9.2 玩家生命值（HP 系统）

玩家在爬塔过程中有一个**全局 HP 条**（区别于单场战斗的 RPM 条）：

- 初始 HP：3 颗心（或 100 HP）
- 战斗失败（RPM 归零）：损失 1 颗心
- HP 归零：Run 结束
- 恢复 HP：商店购买（50G/颗心）、特殊事件奖励

### 9.3 难度递增

| 层级 | 敌人数量 | 敌人强度 | 特殊规则 |
|------|---------|---------|---------|
| **Floor 1** | 1-2 个 | 基础 | 无 |
| **Floor 2** | 2-3 个 | 强化 | 敌人可能有元素攻击环 |
| **Floor 3** | 2-4 个 | 精英 | 敌人可能有特殊被动 |
| **Floor 4（Boss）** | 1 个 Boss | Boss 级 | 三阶段机制 |

## 11. 视觉与音效方向

### 13.1 视觉风格

- **整体风格**：赛博朋克 + 机械美学。陀螺是精密的机械装置，竞技场是高科技斗技场。
- **色彩方案**：深色背景（黑/深蓝）+ 高饱和度元素色（火=橙红，冰=青蓝，电=黄白，毒=绿紫）

### 13.2 音效设计

| 音效类型 | 设计方向 |
|----------|---------|
| **陀螺旋转音** | 高频嗡鸣声，RPM 越高音调越高，RPM 降低时音调下降（实时音高变化） |

| **背景音乐** | 电子音乐 + 机械节拍，战斗时节奏加快，准备回合时舒缓 |

## 附录 B：核心风险与应对

| 风险 | 描述 | 应对方案 |
|------|------|---------|
| **物理模拟复杂度** | 真实陀螺物理（进动/章动）实现难度高 | MVP 阶段用简化物理，后期迭代精度 |

| **Build 多样性** | 某些零件组合过于强势 | 数值迭代 + 挑战模式限制特定零件 |
