## 一、 项目概述

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| **项目代号** | Handle With Panic |
| **中文名称** | 废墟快递 |
| **游戏类型** | 1-4人合作 / 物理模拟喜剧 / 沙盒解谜 |
| **游戏视角** | 3D第一人称 |
| **目标平台** | PC (Steam) |
| **目标受众** | 喜欢合作游戏、派对游戏的玩家群体（18-35岁） |
| **参考游戏** | 《RV There Yet?》、《Overcooked》、《Moving Out》、《Human Fall Flat》 |

### 1.2 核心体验关键词

```
"奇葩道具涌现" + "非对称物理职业" + "特异货物保姆" + "动态地形逃生"
```

### 1.3 游戏一句话描述

玩家扮演废土世界的快递员，在物理法则的"恶意"下，与队友协作将各种奇葩货物安全送达目的地。

### 2.1 循环流程图

```
┌─────────────────────────────────────────────────────────────────┐

│   ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐ │

│         └──────────────────────────────────────────────┘       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 阶段详解

#### 阶段一：接单与入职

**时长：** 30-60秒

| 步骤 | 描述 | 设计意图 |
|------|------|----------|
| 领取订单 | 玩家在黑市墙上查看当日可选的【特异货物】订单 | 提供选择权，增加策略深度 |
| 抢夺工牌 | 玩家通过物理抢夺墙上的"员工工牌"来决定本局的【物理特质职业】 | 创造社交互动，增加趣味性 |
| 职业分配 | 工牌决定职业，不可更换（本局永久） | 强制合作，避免"最优解"固化 |

**设计要点：**
- 工牌抢夺采用物理交互，玩家需要真正"伸手去抢"
- 工牌数量 = 玩家数量，确保每人都有职业
- 允许玩家间协商，但最终由抢夺结果决定

---

#### 阶段二：配装与构筑

**时长：** 60-90秒

| 步骤 | 描述 | 设计意图 |
|------|------|----------|
| 道具池生成 | 系统随机生成本局【公共运输道具池】（3-5件道具） | 增加重玩价值，避免套路化 |
| 路线观察 | 玩家可查看前方路线的预览图/望远镜 | 信息不对称，促进沟通 |
| 战术商讨 | 玩家讨论分工、道具分配、路线选择 | 强化社交体验 |
| 道具选择 | 每位玩家根据职业特点挑选1-2件道具携带 | 策略选择，职业差异化体现 |

**道具池示例：**

| 道具名称 | 功能描述 | 适用场景 |
|----------|----------|----------|
| 毛毯 | 可承载货物，多人协作抬运 | 易碎货物、情绪型货物 |
| 攻城弩 | 发射绳索，可牵引货物或玩家 | 跨越深渊、救援队友 |
| 购物车 | 可装载货物推行，但转向困难 | 平坦地形、大型货物 |
| 弹射板 | 可将货物/玩家弹射到远处 | 快速跨越、高风险高回报 |
| 充气垫 | 落地缓冲，保护货物免受坠落伤害 | 高落差地形 |
| 磁力手套 | 可吸附金属货物，单手搬运 | 金属货物、解放双手 |

---

#### 阶段三：环境对抗与涌现

**时长：** 15-30分钟（根据关卡难度）

**核心挑战维度：**

```
┌────────────────────────────────────────────────────────────┐
│                     环境威胁类型                            │
├──────────────┬──────────────┬──────────────┬──────────────┤
│    地形类    │    动态类    │    环境类    │    生物类    │
├──────────────┼──────────────┼──────────────┼──────────────┤
│ • 深渊断桥   │ • 车流穿梭   │ • 暴风雨     │ • 拦路强盗   │
│ • 陡坡悬崖   │ • 滚石崩塌   │ • 高温熔化   │ • 领地动物   │
│ • 泥沼流沙   │ • 移动平台   │ • 低温冻结   │ • 机械守卫   │
│ • 狭窄通道   │ • 坍塌地板   │ • 辐射区域   │              │
└──────────────┴──────────────┴──────────────┴──────────────┘
```

**第一人称视野限制设计：**

| 设计要素 | 描述 | 合作价值 |
|----------|------|----------|
| 视野范围 | 约90-100度水平视野 | 需要队友报告侧后方情况 |
| 大型货物遮挡 | 搬运大型货物时视野进一步受限 | 需要队友充当"眼睛" |
| 环境干扰 | 暴雨、沙尘暴等会降低能见度 | 强化语音沟通必要性 |
| 声音提示 | 通过环境音效提示危险方向 | 听觉成为重要信息来源 |

---

#### 阶段四：货物状态维护

**货物状态系统：**

```
货物状态 = 基础状态 - (时间衰减 × 货物特性系数) - (碰撞伤害) + (维护行为)
```

| 状态指标 | 影响因素 | 维护方式 |
|----------|----------|----------|
| 完整度 | 碰撞、坠落、挤压 | 小心搬运、使用缓冲道具 |
| 稳定度 | 颠簸、摇晃、倾斜 | 安抚、固定、平稳移动 |
| 特殊状态 | 货物特性（见下文） | 针对性维护行为 |

---

#### 阶段五：送达结算

**结算维度：**

| 维度 | 权重 | 评价标准 |
|------|------|----------|
| 货物完整度 | 40% | 0-100%，低于30%视为失败 |
| 用时 | 30% | 与标准时间对比 |
| 额外目标 | 20% | 是否完成隐藏任务 |
| 团队协作 | 10% | 救援次数、配合评分 |

**奖励内容：**
- 新道具解锁
- 新地形/关卡解锁
- 新货物类型解锁
- 外观装饰品

## 三、 核心机制详解

### 3.1 非对称物理特质系统 (Physics-Based Roles)

**设计理念：** 抛弃传统技能CD设计，用物理法则本身区分职业，强制玩家扬长避短地合作。

#### 职业一：搬运工 (Heavy Lifter)

```
┌─────────────────────────────────────────────────────────────┐
│  【搬运工】—— "稳如泰山，重如泰山"                           │
├─────────────────────────────────────────────────────────────┤
│  物理特性：                                                  │
│  • 质量：标准值的 3.0 倍                                     │
│  • 摩擦力：标准值的 1.5 倍                                   │
│  • 惯性：标准值的 2.5 倍                                     │
├─────────────────────────────────────────────────────────────┤
│  优势：                                                      │
│  ✓ 搬运重物时极度稳定，几乎不会被货物拖拽                    │
│  ✓ 在强风、斜坡上保持平衡能力强                              │
│  ✓ 可以充当"人肉锚点"，用绳索固定队友或货物                  │
│  ✓ 碰撞时对其他物体冲击力大                                  │
├─────────────────────────────────────────────────────────────┤
│  劣势：                                                      │
│  ✗ 移动速度慢，加速/减速都需要更长时间                        │
│  ✗ 容易压塌脆弱地形（木板桥、薄冰面）                        │
│  ✗ 被弹射类道具发射时，抛物线下坠严重，射程短                │
│  ✗ 游泳/漂浮能力弱，容易沉底                                 │
├─────────────────────────────────────────────────────────────┤
│  最佳搭档：跑腿小哥（利用速度差创造战术）                     │
│  避免搭档：另一位搬运工（地形承受力不足）                     │
└─────────────────────────────────────────────────────────────┘
```

**典型使用场景：**
- 搬运超重货物通过强风区域
- 在移动平台上充当稳定器
- 用身体挡住滚石保护货物

```
│  • 惯性：标准值的 0.4 倍                                     │
├─────────────────────────────────────────────────────────────┤
│  优势：                                                      │

│  ✓ 弹射类道具可将其发射到极远位置                            │
│  ✓ 可以踩踏脆弱地形而不损坏                                  │
│  ✓ 游泳/漂浮能力强，可水上作业                               │

```

**典型使用场景：**
- 快速穿越危险区域侦察路线

- 运送轻型急需物品

```
┌─────────────────────────────────────────────────────────────┐

```

**典型使用场景：**
- 持续陪伴情绪型货物防止暴走
- 在高温环境中减缓冰块融化
- 协调团队行动，充当指挥角色

```
├─────────────────────────────────────────────────────────────┤

│  • 摩擦力：标准值                                            │
```

**组合道具示例：**

| 组合方案 | 效果 | 风险 |
|----------|------|------|
| 毛毯 + 弹射板 | 弹射货物时有缓冲保护 | 弹射板可能提前触发 |
| 购物车 + 攻城弩 | 可发射购物车 | 后坐力可能导致翻车 |
| 充气垫 + 磁力手套 | 磁力吸附+落地保护 | 磁力可能失效 |

### 3.2 特异货物系统 (Cargo Affixes)

**设计理念：** 货物具有极端的物理/环境特性，直接改变当前对局的通关逻辑。

#### 货物类型一：情绪型 (Emotional)

```
┌─────────────────────────────────────────────────────────────┐
│  【情绪型货物】—— "会发脾气的好朋友"                         │
├─────────────────────────────────────────────────────────────┤
│  代表货物：活体包裹、宠物笼、神秘宝箱                         │
├─────────────────────────────────────────────────────────────┤
│  特性：                                                      │
│  • 稳定度随颠簸程度快速下降                                   │
│  • 稳定度过低时触发"暴走"状态                                │
│  • 暴走表现：自爆/逃跑/反向移动                              │
│  • 需要持续"安抚"行为维持稳定                                │
├─────────────────────────────────────────────────────────────┤
│  暴走行为表：                                                │
│  ┌─────────┬───────────────────────────────────────┐        │
│  │ 稳定度  │ 行为表现                              │        │
│  ├─────────┼───────────────────────────────────────┤        │
│  │ 80-100% │ 正常，偶尔轻微晃动                    │        │
│  │ 50-79%  │ 频繁晃动，发出警告声                  │        │
│  │ 20-49%  │ 剧烈挣扎，可能挣脱束缚                │        │
│  │ 0-19%   │ 暴走：自爆/逃跑/攻击玩家             │        │
│  └─────────┴───────────────────────────────────────┘        │
├─────────────────────────────────────────────────────────────┤
│  安抚方式：                                                  │
│  • 持续接触（安抚员加成）                                    │
│  • 特定节奏的轻拍                                            │
│  • 使用安抚道具（音乐盒、镇静剂）                            │
└─────────────────────────────────────────────────────────────┘
```

**设计要点：**
- 情绪型货物增加"保姆"玩法深度
- 强制团队分配人手专门照顾货物
- 与"安抚员"职业形成强绑定

---

#### 货物类型二：易碎型 (Fragile)

```
┌─────────────────────────────────────────────────────────────┐
│  【易碎型货物】—— "玻璃心，玻璃身"                           │
├─────────────────────────────────────────────────────────────┤
│  代表货物：古董花瓶、精密仪器、玻璃艺术品                     │
├─────────────────────────────────────────────────────────────┤
│  特性：                                                      │
│  • 严禁大落差坠落（>1米）                                    │
│  • 严禁剧烈碰撞                                              │
│  • 封印弹射类道具的使用                                      │
│  • 完整度直接决定收益                                        │
├─────────────────────────────────────────────────────────────┤
│  损伤阈值：                                                  │
│  ┌─────────────────┬─────────────────────────────┐          │
│  │ 触发条件        │ 损伤量                      │          │
│  ├─────────────────┼─────────────────────────────┤          │
│  │ 轻微碰撞        │ 5-10%                       │          │
│  │ 中等碰撞        │ 20-30%                      │          │
│  │ 剧烈碰撞        │ 50-70%                      │          │
│  │ 坠落(1-2米)     │ 30-50%                      │          │
│  │ 坠落(>2米)      │ 80-100%（可能直接碎裂）     │          │
│  └─────────────────┴─────────────────────────────┘          │
├─────────────────────────────────────────────────────────────┤
│  保护策略：                                                  │
│  • 使用毛毯/充气垫包裹                                       │
│  • 避免弹射、快速移动                                        │
│  • 选择平坦路线                                              │
│  • 多人协作平稳搬运                                          │
└─────────────────────────────────────────────────────────────┘
```

**设计要点：**
- 限制"暴力速通"策略
- 强调精细操作和团队配合
- 与"跑腿小哥"职业形成张力（速度快但易出事）

#### 货物类型三：环境反馈型 (Environmental)

```
┌─────────────────────────────────────────────────────────────┐

│  • 接触水会熄灭（失败）                                      │
│  • 可照亮黑暗区域                                            │
│  • 可能点燃周围可燃物                                        │
├─────────────────────────────────────────────────────────────┤
│  子类型C - 植物类：                                          │
│  • 需要特定光照/水分条件                                     │
│  • 环境不适会枯萎                                            │
│  • 可能在途中生长变大                                        │
└─────────────────────────────────────────────────────────────┘
```

**设计要点：**
- 将环境因素纳入核心玩法
- 改变路线选择逻辑
- 增加时间压力

#### 货物类型四：伤害型 (Hazardous)

```
┌─────────────────────────────────────────────────────────────┐
│  【伤害型货物】—— "烫手山芋，接力传递"                       │
├─────────────────────────────────────────────────────────────┤
│  代表货物：辐射源、剧毒化学品、诅咒神器                       │
├─────────────────────────────────────────────────────────────┤
│  特性：                                                      │
│  • 距离越近/携带越久，玩家受到伤害                           │
│  • 产生操作负面状态（屏幕扭曲、反向键、减速）                 │
│  • 必须在队友间接力传递                                      │
│  • 有"安全距离"和"安全时间"概念                              │
├─────────────────────────────────────────────────────────────┤
│  负面状态表：                                                │
│  ┌─────────────┬───────────────────────────────────┐        │
│  │ 暴露时间    │ 负面效果                          │        │
│  ├─────────────┼───────────────────────────────────┤        │
│  │ 0-5秒       │ 无明显效果                        │        │
│  │ 5-10秒      │ 视野边缘轻微扭曲                  │        │
│  │ 10-15秒     │ 移动速度-20%，操作延迟            │        │
│  │ 15-20秒     │ 控制键随机反向，持续掉血          │        │
│  │ >20秒       │ 大幅掉血，可能直接倒地            │        │
│  └─────────────┴───────────────────────────────────┘        │
├─────────────────────────────────────────────────────────────┤
│  应对策略：                                                  │
│  • 设置接力点，定时交接                                      │
│  • 使用防护道具（铅衣、防毒面具）                            │
│  • 安排"替补"队员随时接手                                    │
│  • 利用道具远程搬运                                          │
└─────────────────────────────────────────────────────────────┘
```

**设计要点：**
- 强制团队协作和沟通
- 创造紧张感和节奏感
- 与"搬运工"职业形成张力（稳定但暴露时间长）

**动态威胁示例：**

| 暴风雨 | 随机触发 | 寻找掩体/固定货物 | 货物被吹走 |

## 九、 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| 物理Bug过多 | 体验受损 | 预留充足测试时间，建立Bug反馈机制 |
