# 实施计划：竞技盘碗形倾斜增强

- [ ] 1. 扩展 ARENA 常量，新增碗形物理与视觉参数
   - 在 `ARENA` 对象中新增 `bowlForceQuadratic`（二次项系数，建议初始值 0.000003）
   - 调高 `bowlForce` 线性系数（建议从 0.0008 提升至 0.0015）
   - 新增视觉参数 `perspectiveRatio`（椭圆纵轴压缩比，建议 0.65）、`contourCount`（等高线数量，建议 4）
   - _需求：3.5、4.2_

- [ ] 2. 重写竞技场渲染函数，实现碗形透视视觉
   - 将竞技盘外轮廓从正圆改为椭圆（`radiusX = ARENA.radius`，`radiusY = ARENA.radius * perspectiveRatio`）
   - 绘制至少 4 层同心椭圆径向渐变，颜色从边缘深色（`#3a2a1a`）到中心浅色（`#c8a87a`）
   - 在 12 点方向绘制高光弧（白色半透明），在 6 点方向绘制阴影弧（黑色半透明）
   - 绘制 `contourCount` 条同心椭圆等高线（半透明白色，间距均匀）
   - 在最外圈绘制 rim highlight（亮色椭圆描边，线宽 3px）
   - _需求：1.1、1.2、1.3、1.4、1.5_

- [ ] 3. 实现陀螺渲染的透视映射
   - 在渲染陀螺前，根据陀螺距中心距离计算透视偏移量（`yOffset = dist * 0.15 * perspectiveRatio`）
   - 根据距离计算尺寸缩放（`scale = 1 - dist / ARENA.radius * 0.15`，最小 0.85）
   - 距中心 < 30px 时不施加偏移和缩放，保持原始坐标
   - _需求：2.1、2.2、2.3_

- [ ] 4. 增强碗形向心物理（非线性向心力）
   - 在 `physicsStep()` 的向心力计算段，将原线性公式替换为：`force = bowlForce * dist + bowlForceQuadratic * dist²`
   - 低 RPM（< 40%）时向心力系数乘以 1.8（原 1.5）
   - 保留 `bowlForceDeadZone`（30px）死区逻辑不变
   - _需求：3.1、3.2、3.3、3.4_

- [ ] 5. 验证视觉与物理边界一致性
   - 确认椭圆 `radiusX` 与 `ARENA.radius` 使用同一变量，物理碰撞边界不变
   - 在浏览器中测试：将陀螺放置于边缘，观察 3 秒内是否向中心移动 > 30px
   - 检查等高线间距与向心力强度的感知是否匹配（视觉坡度陡的区域陀螺加速明显）
   - _需求：4.1、4.2_
