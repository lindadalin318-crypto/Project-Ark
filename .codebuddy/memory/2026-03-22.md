# 2026-03-22 工作记录

## Bloom Burst 不可见修复
- **根因**: Main Camera 的 `UniversalAdditionalCameraData.renderPostProcessing` 为 `false`，URP 跳过所有后处理
- **修复**: 通过 MCP 开启 `renderPostProcessing = true` 并保存场景
- **防御**: 在 `BoostTrailView.Awake()` 新增 `ValidateCameraPostProcessing()` 检查
- **踩坑**: Bloom 是后处理效果，粒子系统不受影响 → 症状表现为"部分 VFX 正常，Bloom 不工作"，容易误判为代码 bug

## 全局 DoF 导致画面模糊
- **根因**: `GlobalVolumeProfile.asset` 中 DepthOfField 默认启用（Bokeh, focusDistance=5, focalLength=50），开启 Post Processing 后全画面模糊
- **修复**: 将 DoF 的 `active` 设为 `0`（默认关闭）。编织态需要的 DoF 由 `WeavingStateTransition.cs` 动态控制，不受影响
- **经验**: 2D 俯视角游戏不应全局常开 DoF，所有 Sprite 在同一 Z 平面，DoF 只会产生不可控模糊
