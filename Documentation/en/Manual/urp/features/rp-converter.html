    Render Pipeline Converter window for URP reference

    After you configure your project to use URP, you can use the Render PipelineA series of operations that take the contents of a Scene, and displays them on a screen. Unity lets you choose from pre-built render pipelines, or write your own. More info
    See in Glossary     Converter to convert assets made for a Built-In Render Pipeline project to assets compatible with the Universal Render Pipeline (URP).

    To open the Render Pipeline Converter window, go to Window > Rendering > Render Pipeline Converter. For more information on how to use the Render Pipeline Converter for URP, refer to Convert assets and quality levels from the Built-In Render Pipeline to URP or Convert 2D assets from the Built-In Render Pipeline to URP.

    Warning: Using the Render Pipeline Converter overwrites several files in your project folder. These can’t be restored after Unity overwrites them. Before you start this task, back up any files you don’t want to lose.

    Pipeline Converter tab

    Property Description
    Source Pipeline Sets the pipeline you want to convert your assets from.
    Target Pipeline Sets the pipeline you want to convert your assets to. The options are Universal Render Pipeline (Universal Renderer) and Universal Render Pipeline (2D Renderer).
    Rendering Settings Converts the quality levels in the Built-in Render Pipeline project to equivalent properties in the URP assets.

    This converter is available only when Target Pipeline is Universal Render Pipeline (Universal Renderer).
    Animation Clip Converts animation clips.

    This converter is available only when Target Pipeline is Universal Render Pipeline (Universal Renderer) and if the project contains animation clips that affect the properties of materials.
    Material Reference Converter Converts references to Built-In Render Pipline materials to references to URP materials.

    This converter is available only when Target Pipeline is Universal Render Pipeline (Universal Renderer).
    Material Shader Converter Converts prebuilt shaders for Built-In Render Pipeline to prebuilt shaders in URP. This converter doesn’t support custom shaders. For more information on how to convert custom shaders for URP, refer to Upgrade custom shaders for URP compatibility.

    This converter is available only when Target Pipeline is Universal Render Pipeline (Universal Renderer).
    Post-Processing Stack v 2 Converts Post-Processing Stack v2 assets to URP post-processing assets.

    This converter is available only when Target Pipeline is Universal Render Pipeline (Universal Renderer) and the Post-Processing Stack v2 package is installed.
    Material and Material Reference Upgrade Converts all materials and references to materials from Built-In Render Pipeline to URP 2D.

    This converter is available only when Target Pipeline is Universal Render Pipeline (2D Renderer).

    Pipeline Tools tab

    Property Description
    Parametric to Freeform Light Upgrade Converts all deprecated parametric lights to freeform lights. For more information refer to Light 2D component reference for URP.
    Convert Built-in and URP (Universal Renderer) Materials to MeshThe main graphics primitive of Unity. Meshes make up a large part of your 3D worlds. Unity supports triangulated or Quadrangulated polygon meshes. Nurbs, Nurms, Subdiv surfaces must be converted to polygons. More info
    See in Glossary     2D-Lit-Default    		 Converts all 3D materials to URP 2D materials.
