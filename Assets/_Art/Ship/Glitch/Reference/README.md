## Usage in Project Ark

| File | Size | Usage |
|------|------|-------|
| `GrabGun_Base_9.png` | 430×430 | Ship_Sprite_Solid + Ship_Sprite_Liquid (shared) |
| `GrabGun_Base_8.png` | 430×430 | Ship_Sprite_HL (highlight layer) |
| `GrabGun_Back_3.png` | 186×96  | Ship_Sprite_Back (thruster/back layer) |

## Import Settings

All textures should be configured with:
- **PPU**: 320
- **Pivot**: Center
- **Filter Mode**: Point (no filter)
- **Sprite Mode**: Single
