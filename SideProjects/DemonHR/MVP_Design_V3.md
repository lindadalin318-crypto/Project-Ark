# 项目代号：《魔王的人力资源部》 (Demon HR: Hero Ruination) — V3.1 双面间谍版

## 版本变更摘要 (V2.1 → V3.0 → V3.1)

V3.0 保留了 V2.1 的所有核心设定（战役季度制、群聊主战场、社交节点操纵、中间胜态链、招安路线、角色持续化），并针对 V2.1 评审中暴露的四个结构性问题进行了系统性补强：

| 问题 | V2.1 状态 | V3.0 解决方案 |
|------|-----------|--------------|
| 操作之间缺乏组合性 | 5 种操作各自独立 | 新增 **操作链系统 (Operation Chain)**：引入"布置→触发→放大"三拍节奏，卡牌之间产生乘法 synergy |
| Admin 权限无代价 | 每次操作必定成功，无风险 | 新增 **暴露风险系统 (Exposure Risk)**：每次操作积累可疑度，勇者方会升级反制 |
| 全知视角缺乏博弈 | 玩家默认看到所有信息 | 新增 **主动情报系统 (Active Intel)**：情报带宽有限，玩家必须选择监听什么 |
| 右侧归因模糊 | 2-5 秒自动战斗难以归因 | 重设计右侧为 **事故复盘系统 (Incident Replay)**：先给结果，再可选展开因果链 |

V3.1 在 V3.0 基础上新增两个重大设计模块：

| 新增模块 | 解决的问题 | 核心内容 |
|----------|-----------|---------|
| **MICE 情报官操作体系** | V3.0 招安系统深度不足，策反过程缺少经营感 | 引入真实情报学的 MICE 四维策反模型（金钱/理念/胁迫/自尊）+ 资产发展周期（发现→评估→培养→策反→运营） |
| **双面立场系统 (Dual Stance)** | 系统只提供破坏工具会把角色框定为"待攻破的目标"，阻碍共情 | 同一套卡牌可反向使用（保护而非破坏），引入魔王忠诚度计量器，让"保护谁、牺牲谁"成为与破坏同等深度的策略线 |

全部系统模块清单：

- **操作链系统 (Operation Chain)**：布置→触发→放大三拍节奏，卡牌之间产生乘法 synergy
- **暴露风险系统 (Exposure Risk)**：每次操作积累可疑度，勇者方会升级反制
- **主动情报系统 (Active Intel)**：情报带宽有限，必须选择监听什么
- **事故复盘系统 (Incident Replay)**：先给结果，再可选展开因果链
- **优雅度评价 (Elegance Rating)**：同一目标有粗暴和精妙两种达成方式
- **连锁反应 / 棋子化 (Cascade & Pawn)**：被操纵的勇者自发产生冲突节点
- **渗透者系统 (Infiltrator)**：Q2+ 解锁，安插卧底进行角色扮演式操纵
- **卡牌体系重设计**：四大家族 × 三个节奏层级 × 双面使用，支持构筑流派与 combo 路线
- **人格交互矩阵 (Personality Matrix)**：人格之间有主动化学反应
- **MICE 情报官操作体系**：四维策反模型 + 资产发展周期 + 策反后运营
- **双面立场系统 (Dual Stance)**：卡牌双面使用 + 魔王忠诚度 + 保护与破坏的双向博弈

## MVP 极简描述

**游戏类型**：单机 RPG / 战役式 Deckbuilding / 社交工程模拟 / 多结局  
**视觉风格**：复古像素风 + 现代职场通讯软件 UI 包装  
**核心体验**：潜入勇者通讯后台，通过情报监听、信息操纵与组织手段，从内部瓦解冒险小队——或将其关键成员策反至魔王阵营  
**一句话卖点**：你不是打败勇者的人，你是让勇者自己打败自己的人

## 一、产品定位 (Product Positioning)

本作不是纯《杀戮尖塔》式的高随机爬塔，也不是纯《欺诈之地》式的重剧情对话冒险，而是一个 **固定战役结构 + 每季度局内构筑 + 角色跨季度持续 + 社交工程为核心博弈** 的单机战役型卡牌 RPG。

### 1.1 核心结构定义

- **整个游戏**：一个完整的 `Campaign`（战役）
- **每个季度**：一个固定目标小队的 `Run`（局）
- **每个季度内**：一张小型地图，由多个遭遇、事件、休整点组成

### 1.2 核心卖点

- 玩家不是正面击败勇者，而是作为魔王军 HR 在幕后"做局"
- 操纵的对象不是血量和护甲，而是承诺、信任、压力、权威和信息
- 勇者角色带着创伤、关系与履历跨季度持续存在
- 同一个目标有粗暴和精妙两种达成方式——精妙路径是本作的技巧天花板
- 游戏存在多结局，玩家的立场会在"忠于魔王"与"共情勇者"之间逐步分化

### 1.3 竞品差异化

| 维度 | 杀戮尖塔 | 欺诈之地 | 本作 |
|------|---------|---------|------|
| 核心战斗 | 打怪出牌 | 对话说服 | 社交工程 + 信息操纵 |
| 敌人存在感 | 机制载体 | 有性格的对话对象 | 会自我修复、会反侦察的完整社会组织 |
| 随机性 | 高（每局全新地图） | 中 | 低（固定战役，局内构筑变化） |
| 跨局连续性 | 无 | 低 | 高（角色/关系/创伤/招安跨季度持续） |
| 玩家权力幻想 | 强力战士 | 巧舌如簧的说客 | 幕后操盘的情报主管 |

## 二、战役结构 (Campaign Structure)

### 2.1 总体思路

游戏由 `Q1-Q4`（标准）或 `Q1-Q5`（完整版含终局）组成。每个季度都是一个相对固定的 Run，玩家需要在 HR 资源耗尽前瓦解当前季度负责的小队。

### 2.2 季度原型

- **Q1：初次出征勇者小队**
  - 作用：教学季度，建立"侦察 → 监听 → 干预 → 执行"的四阶段循环
  - 特点：成员年轻、性格外露、关系简单，无反侦察能力
  - 系统解锁：基础五操作（撤回/伪造/延迟/放大/泄露），暴露风险入门
- **Q2：皇家骑士团**
  - 作用：引入反侦察与渗透者机制
  - 特点：可能包含 Q1 存活成员；队伍更团结、领导力更强
  - 系统解锁：渗透者系统，操作链 combo，勇者反制升级（Awareness Level 1）
- **Q3：精英远征队**
  - 作用：将小队矛盾升级为制度矛盾、荣誉矛盾与功绩矛盾
  - 特点：内部派系明确，存在预设裂痕但也有更强的自愈能力
  - 系统解锁：高级行政操作，双卧底，勇者反制升级（Awareness Level 2）
- **Q4：王国直属讨魔队 / 圣征军**
  - 作用：引入王国最强、最稳定、最难瓦解的官方精锐
  - 特点：高凝聚力、高纪律、有专职反间角色
  - 系统解锁：全卡池开放，终极 combo 路线
- **Q5：终局季度（可选）**
  - 作用：根据前几个季度的存活者、招安者与重大事件，正式导向多结局

### 2.3 失败结构

- 若某季度失败，**直接重打该季度**，不回退到前一季度
- 重打时保留已获得的卡牌和权限，但队伍状态重置
- 设计原则：季度是清晰的章节边界，失败惩罚要存在但不能摧毁叙事连续性

### 2.4 季度长度

每个季度为一张小型地图，包含：

- 普通遭遇（3-5 个）
- 固定叙事事件点（1-2 个）
- 休整 / 牌组调整点（1-2 个）
- 季度 Boss 遭遇或终局事件（1 个）

单季度预期游玩时长：45-90 分钟。

## 三、核心视觉与信息流 (The Group Chat Interface)

玩家作为魔王军 HR，拥有勇者公会内部通讯的"后台管理员权限"。

### 3.1 双屏表达

- **左侧主屏（70%）**：勇者小队通讯群聊 + 情报面板
  - 展示群聊消息流、提炼后的社交节点、关系/状态板
  - 这是玩家的**主决策界面**
- **右侧副屏（30%）**：事故复盘终端
  - 不再是"实时自动战斗演出"，而是**结果先行 + 可选因果回放**
  - 核心职责：把左侧操纵的后果以可理解、可复盘的方式呈现

### 3.2 信息展示原则

- 文本负责叙事乐趣与角色塑造
- 图标、标签和状态 Token 负责策略判断
- 玩家平时看摘要信息做决策，想看乐子时再阅读完整群聊
- 左侧是主决策界面，右侧是后果验证与爽感界面

### 3.3 右侧副屏：事故复盘系统 (Incident Replay)

V2.1 的"2-5 秒高速自动战斗"存在归因模糊的根本缺陷。V3.0 重新定义右侧为**事故复盘终端**：

#### 执行阶段的展示流程

1. **结果速报**（0.5 秒）
   - 先直接给出本回合结果摘要：谁受伤了、谁的承诺违约了、谁和谁吵起来了
   - 用状态变化图标一目了然地展示

2. **因果链回放**（可选展开，每条 2-3 秒）
   - 玩家可点击任意事故节点，展开一条 2-3 步的因果链：
   - 例如：`[你撤回了牧师的治疗承诺] → [牧师在执行时犹豫了一拍] → [战士在缺少治疗的窗口期被打到危险血线] → [战士的暴躁人格触发，向牧师发难]`
   - 这条链清晰展示了"玩家操作 → 角色行为改变 → 战斗事故 → 社交后果"的完整因果

3. **关键时刻特写**（自动触发，1-2 秒）
   - 当出现死亡、叛逃、招安机会、团队崩溃等重大事件时，自动播放一段像素风特写动画
   - 配合角色台词气泡，强化戏剧感

#### 设计原则

- **归因清晰优先于视觉表演**：玩家必须能准确知道"这个事故是我的哪步操作导致的"
- **结果先行，过程可选**：不强迫玩家看完整回放，但好奇的玩家可以深挖
- **因果链本身就是奖励**：看到自己三步前埋下的伏笔终于引爆，这种因果快感是本作核心爽点

## 四、单季度核心循环 (Quarter Run Loop)

每个季度的目标：**在资源耗尽前，使目标小队失去继续推进任务的能力。**

### 4.1 单回合四阶段

V3.0 将 V2.1 的三阶段扩展为四阶段，将"情报获取"从"被动接收"改为"主动选择"：

#### 阶段一：侦察部署 (Recon Phase)

- 玩家分配本回合的**情报带宽**（详见 4.2 主动情报系统）
- 决定监听哪些频道、对哪些角色进行深度扫描
- 本阶段不消耗预算，但带宽分配决定了后续能获得多少信息

#### 阶段二：情报监听 (Wiretapping Phase)

- 根据侦察部署的结果，系统展示对应频道的群聊内容
- 未被监听的频道只显示模糊摘要或"[加密通讯]"
- 系统从可见消息中提炼出社交节点（承诺、命令、抱怨、请求等）
- 暴露当前可见的压力值、信任波动与潜在爆点
- **关键差异**：你可能因为带宽不足而漏掉某些关键信息

#### 阶段三：HR 干预 (Intervention Phase)

- 玩家消耗预算与手牌进行干预
- 干预分为三类（详见第五章 卡牌系统）：
  - **信息操作卡**：撤回、伪造、延迟、泄露
  - **社交操作卡**：放大、栽赃、公开引用、舆论污染
  - **行政操作卡**：审计、人事调动、预算削减、绩效篡改
- 每次操作会积累**暴露值**（详见 4.3 暴露风险系统）
- 支持**操作链 combo**（详见 4.5 操作链系统）

#### 阶段四：执行与复盘 (Execution & Replay Phase)

- 勇者小队执行本回合行动
- 右侧事故复盘终端展示结果速报 + 可选因果链
- 出现 Reaction Window 时允许少量即时干预
- 结算后进入**战后余波**：角色在群聊中对事故进行反应，生成下一回合的情绪状态

### 4.2 主动情报系统 (Active Intel)

V2.1 中玩家默认看到所有群聊内容，这使得决策退化为优化题。V3.0 引入情报带宽限制：

#### 情报带宽

- 每回合玩家拥有固定的**带宽点数**（初始 3 点，可通过升级增加）
- 带宽分配选项：

| 频道类型 | 带宽消耗 | 信息质量 |
|----------|---------|---------|
| 群聊（公开频道） | 0（免费） | 公开消息全文 + 自动提炼节点 |
| 私聊（A↔B） | 1 点 / 对 | 私聊全文，可能发现隐藏矛盾或密谋 |
| 深度扫描（单角色） | 2 点 | 该角色的内心独白 / 真实想法 / 隐藏人格触发条件 |
| 通讯元数据 | 1 点 | 谁在和谁频繁私聊、谁被孤立、消息响应速度 |

#### 信息缺失的后果

- 未监听的私聊显示为 `[加密通讯 — 内容未知]`
- 你可能不知道两个角色在私下里已经和好了（导致你对公开裂痕的判断过时）
- 你可能不知道某个角色正在秘密联络公会总部请求换队
- 这些"信息盲区"让每一局都有信息不完美性，防止退化为纯优化题

#### 情报带宽的策略价值

- **广撒网**：监听多个私聊频道，获得更完整的信息图景，但缺少任何一个角色的深层洞察
- **深聚焦**：对关键角色做深度扫描，精准击穿其防线，但可能遗漏其他角色的暗中行动
- **元数据分析**：不看内容但看模式，发现隐藏的社交结构变化

### 4.3 暴露风险系统 (Exposure Risk)

V2.1 中 Admin 操作无代价。V3.0 引入暴露系统，让每次操作都成为风险-收益权衡：

#### 可疑度 (Suspicion)

- 每次使用管理员权限或卡牌操作，会积累**可疑度**（0-3 点/次）
- 可疑度在季度内持续累积，不会自动衰减
- 不同操作的可疑度成本不同：

| 操作类型 | 可疑度 | 说明 |
|----------|--------|------|
| 延迟消息 | +0 | 网络延迟很常见，几乎不会被怀疑 |
| 已读不回 | +0 | 无法追溯到外部干预 |
| 放大/公开引用 | +1 | 略显刻意，但可解释为系统推送 |
| 撤回消息 | +1 | 发送者可能察觉"我没撤回啊？" |
| 伪造消息 | +2 | 高风险，内容风格不匹配可能被识破 |
| 禁言 | +2 | 明显的外部干预 |
| 篡改绩效/审计 | +3 | 行政操作留有痕迹 |

#### 勇者觉察等级 (Awareness Level)

当可疑度达到阈值时，勇者队伍整体进入更高的觉察等级：

- **Level 0（默认）**：勇者对通讯安全无感知，所有操作正常生效
- **Level 1（可疑度 ≥ 10）**：
  - 多疑型角色开始使用私聊代替群聊讨论敏感话题
  - 部分群聊消息变得含糊（"你知道的那件事"代替具体计划）
  - 玩家对公开群聊的信息获取量下降约 30%
- **Level 2（可疑度 ≥ 20）**：
  - 队伍中可能出现"反间意识"——角色开始测试通讯是否被窃听（发送诱饵消息）
  - 如果玩家对诱饵消息做出反应（撤回/伪造），可疑度额外 +5
  - 伪造消息的成功率下降（角色会互相确认）
- **Level 3（可疑度 ≥ 30）**：
  - 队伍进入"通讯管制"：关键战术讨论转为线下面谈，群聊中只剩日常闲聊
  - 玩家的信息操作卡效力减半
  - 但同时队伍因为"知道被监视"而产生额外压力（+压力给全队）
  - **如果玩家安插了渗透者**，渗透者面临被排查的风险

#### 暴露系统的策略意义

- 创造了**激进 vs 保守**的核心张力：激进操作推进快但暴露高，保守操作安全但可能被勇者自我修复
- 鼓励"优雅"的操作方式：低暴露的精妙 combo 比高暴露的粗暴操作更有价值
- 不同季度的暴露预算不同：Q1 阈值高（勇者新手不警觉），Q4 阈值低（精英队伍有反间意识）

### 4.4 左侧群聊：主战场 (Social Combat Layer)

左侧通讯群聊不是叙事包装，而是本作真正的**主战场**。玩家进行的是一场围绕承诺、信任、压力、权威和信息传递的**社交工程战**。

#### 核心博弈对象

- **承诺**：社交契约（"我给你加盾""你先扛我第二拍到"），是右侧事故的主要因果来源
- **信任**：决定角色是否相信队友、是否愿意照计划行动
- **压力**：压力越高，角色越容易误读、失控、崩溃或进入可招安状态
- **权威**：话语权等级差异，影响命令效力与背书权重
- **注意力 / 信息传递**：谁看到了什么、谁的消息被淹没、哪句话被置顶、哪条私聊被泄露

#### 社交节点

群聊的基础战斗单位是**可操作的社交节点**：

- 一个治疗承诺
- 一个冲锋命令
- 一次资源分配
- 一条抱怨
- 一次求救
- 一条私聊泄露

玩家干预的对象是这些节点本身，节点被操纵后反向改变人物关系与执行结果。

#### 中间胜态

左侧群聊有自己的"战果"层级，从轻到重：

| 胜态 | 含义 | 效果 |
|------|------|------|
| **违约** | 关键承诺未兑现 | 事故概率上升，信任开始受损 |
| **错位** | 角色行动节奏不再统一 | 战斗效率下降，压力上升 |
| **失声** | 重要信息未能传达给关键成员 | 角色做出错误决策 |
| **甩锅** | 事故责任集中到某人身上 | 被甩锅者压力飙升，其他成员信任下降 |
| **孤立** | 角色被团队边缘化 | 孤立者战斗意志下降，进入可招安前置 |
| **派系** | 队伍分裂为两个对立阵营 | 凝聚力加速下降，内部命令可能被抵制 |
| **动摇** | 角色开始怀疑是否还应继续 | 接近可招安状态 |
| **可招安** | 角色不再把魔王阵营视为绝对敌人 | 可使用招安卡牌 |

#### 勇者方的反制

勇者队伍具备自我修复能力，形成真正的博弈：

- **澄清**：解释误会、否认伪造、恢复消息真实性
- **背书**：队友作证修复被破坏的信任
- **安抚**：压下争吵、降低压力、阻止内讧升级
- **重新分工**：计划被打乱后快速重建协作秩序
- **团建**（V3.0 新增）：在休整点进行主动修复，恢复凝聚力
- **反间排查**（V3.0 新增）：在 Awareness Level 2+ 时主动调查通讯异常

玩家的核心乐趣：**抢在勇者完成修复前，把矛盾推过临界点。**

#### 连锁反应与棋子化 (Cascade & Pawn)

**V3.0 新增核心机制。**

当一个角色因为你的操纵而产生强烈负面情绪时，该角色可能自发地在群聊中产生新的冲突节点：

- 被甩锅的战士可能主动在群聊中指责法师 → 生成新的 `抱怨` 节点
- 压力过高的牧师可能拒绝下一次治疗承诺 → 生成新的 `拒绝` 节点
- 被孤立的盗贼可能公开要求重新分配战利品 → 生成新的 `资源冲突` 节点

**这些角色生成的节点不会增加你的可疑度**（因为它们是角色自发行为），但你可以对这些新节点继续施加操作。

这就形成了本作的终极策略循环：

```
你的操作 → 角色情绪变化 → 角色自发产生冲突 → 你在新冲突上继续操作 → 更大的连锁反应
```

**棋子化**是连锁反应的高级形态：当一个角色的特定情绪（愤怒/猜疑/贪婪）持续处于高位时，该角色会变成半自动的"棋子"——每回合固定产生 1-2 个与其情绪对应的冲突节点，无需你额外操作。

**棋子化的策略意义**：
- 减少你每回合需要亲自操作的次数 → 节省预算和暴露
- 使冲突看起来完全"有机" → 不增加可疑度
- 但棋子化角色可能失控（产生你不想要的冲突，或自行离队）

#### 优雅度评价 (Elegance Rating)

**V3.0 新增核心机制。**

每回合结算时，系统会评估玩家操作的"优雅度"——衡量的不是你造成了多少伤害，而是你的操纵有多精妙：

| 等级 | 定义 | 条件示例 | 奖励 |
|------|------|----------|------|
| **粗暴 (Brute)** | 直接使用高暴露操作达成目标 | 伪造命令 + 禁言反对者 | 目标达成，但暴露 +4-6 |
| **务实 (Pragmatic)** | 使用中等暴露操作配合基础 combo | 撤回承诺 + 放大抱怨 | 目标达成，暴露 +2-3 |
| **精妙 (Subtle)** | 使用低暴露操作通过多步链条达成目标 | 延迟消息 + 利用角色自发冲突 + 在余波中顺势推进 | 目标达成，暴露 +0-1，额外卡牌奖励 |
| **杰作 (Masterwork)** | 仅用 1-2 次操作触发连锁反应，角色自行走向崩溃 | 一次精准泄露 → 角色棋子化 → 角色自行摧毁团队 | 目标达成，暴露 +0，解锁稀有卡牌 |

**优雅度的设计意义**：
- 让新手和高手面对同一局面时有截然不同的解法
- 为"重打同一季度"提供挑战价值——这次能用更优雅的方式通关吗？
- 与暴露系统联动：优雅操作的低暴露为后续回合保留了更多操作空间

#### 群聊信息的三层结构

- **第一层：聊天流** — 角色魅力、代入感、戏剧性
- **第二层：提炼后的社交节点** — 一眼可读的策略信息
- **第三层：关系 / 状态板** — 长期变量（信任、压力、权威、创伤、可招安性）

设计原则：**文本负责好看，结构化信息负责能玩。**

### 4.5 操作链系统 (Operation Chain)

**V3.0 核心新增系统。**

V2.1 的 5 种操作各自独立，缺乏组合深度。V3.0 引入**三拍节奏**的操作链：

#### 三拍节奏：布置 → 触发 → 放大

| 节奏层 | 说明 | 卡牌特征 |
|--------|------|----------|
| **布置 (Setup)** | 改变某个节点的状态，创造后续操作的条件 | 低费用、低暴露、单独效果弱 |
| **触发 (Trigger)** | 利用已布置的条件，引爆一次有意义的事故 | 中费用、中暴露、需要特定前置 |
| **放大 (Amplify)** | 在事故发生后扩大影响，将个人事故转化为团队危机 | 高费用、高暴露、效果翻倍但需要时机 |

#### 操作链示例

**链条 1：信任崩塌链**
1. [布置] `延迟` 牧师的治疗承诺（牧师的承诺比战士的冲锋命令晚到 → 战士不知道牧师会支援）
2. [触发] 执行阶段：战士冲锋后缺少治疗 → 触发事故 → 战士对牧师产生不信任
3. [放大] 战后余波中，对战士的抱怨使用 `公开引用` → 不信任从私人矛盾升级为公开对质

**链条 2：身份伪装链**
1. [布置] `泄露` 法师的一条私聊（法师曾私下抱怨队长指挥能力差）
2. [触发] `伪造` 一条"队长看到了泄露内容"的系统通知 → 法师以为自己已经被队长知晓
3. [放大] 法师因恐惧主动在群聊中辩解 → 反而让更多人知道了这件事（角色自发产生的连锁节点）

**链条 3：替罪羊链**
1. [布置] `审计` 盗贼的履历（暴露其过往黑历史）
2. [触发] 当战斗出现任何事故时，使用 `栽赃` 将责任引向盗贼
3. [放大] 队伍其他成员看到审计结果后自然倾向于相信"盗贼果然不可靠"→ 盗贼被孤立

#### 操作链的设计价值

- **两张牌 > 两张牌之和**：combo 产生的效果显著大于单独使用
- **鼓励提前规划**：布置阶段的卡牌单独打效果很弱，但如果你能在下回合触发它，价值翻倍
- **创造"忍住不出"的 tempo 决策**：手里有一张触发卡但前置条件还没布置好——是现在打出一个弱效果，还是攒到下回合打一套完整 combo？
- **构筑方向**：玩家可以围绕某类操作链构建牌组（信任崩塌流、替罪羊流、信息战流）

### 4.6 Reaction Window（反应窗口）

执行阶段允许玩家继续干预，但必须通过**事件窗口**触发。

#### 触发原则

- 只在关键节点出现
- 每次提供 2-3 个直观的反应选择
- 目的是放大戏剧性和补刀，而不是重做一遍主策略判断

#### 典型触发时机

- 战士即将冲锋 / 牧师准备治疗
- 某成员血量低于阈值
- 两名成员的信任跌破阈值
- 角色进入可招安状态
- 计划阶段埋下的"布置"条件被满足

#### 反应窗口中的干预

- **环境卡**：增援呼叫、地板塌陷、封门、照明故障
- **反应卡**：断线、误报、恐吓广播、人事冻结
- **条件触发卡**：招安话术、火上浇油、补刀舆论

#### 设计禁区

- 不应要求玩家阅读大量文本
- 不应让玩家重新做完整策略规划
- 不应把执行阶段扩张成第二套卡牌主循环

### 4.7 胜利条件 (Victory Conditions)

| 条件 | 描述 | 评价 | 战役影响 |
|------|------|------|---------|
| **团灭 (Wipeout)** | 全员 HP 归零 | 最低评价 | 角色永久死亡，后续季度无法再出现 |
| **解散 (Disband)** | 团队凝聚力归零 | 较高评价 | 角色存活但关系破裂，可能在后续季度以敌对/中立身份出现 |
| **招安 (Recruitment)** | 关键成员被策反至魔王阵营 | 最高评价 | 招安角色在后续季度提供情报/卡牌/权限 |

新增评价维度：
- **优雅度总分**：整个季度的操作优雅度累计，影响奖励卡牌品质
- **暴露控制**：季度结束时的可疑度越低，下一季度的初始带宽越高

### 5.3 卡牌经济

#### 预算 (Budget)

- 每回合获得固定预算（初始 3 点，可随权限成长提升）
- 未使用的预算**不结转**下回合（鼓励每回合都做有意义的操作）
- 某些事件（成功招安、优雅评价高）可获得额外预算

#### 手牌

- 每回合从牌库抽取手牌（初始抽 4 张）
- 未使用的手牌保留到下回合（但手牌上限为 7 张）
- 地图休整点可进行删牌/升级牌/获取新卡

#### 卡牌升级

部分卡牌可升级为更强版本：

- `撤回` → `无痕撤回`（暴露从 +1 降到 +0）
- `伪造` → `深度伪造`（可模拟目标角色的语言风格，降低被识破概率）
- `挑拨` → `离间计`（同时影响双方，而不只是单方面误解）
- `栽赃` → `完美栽赃`（自动附带伪造证据，无需额外 combo）

### 5.4 构筑流派

卡组构筑的方向不是"取最强卡"，而是围绕特定打法路线选择 combo 配件：

| 流派 | 核心思路 | 关键卡牌 | 优势 | 劣势 |
|------|---------|---------|------|------|
| **信息战 (Disinfo)** | 通过伪造和延迟制造认知混乱 | 伪造、延迟、撤回、窃听升级 | 暴露低、连锁性强 | 需要精确时序，门槛高 |
| **替罪羊 (Scapegoat)** | 集中火力孤立一人 | 审计、栽赃、人事背锅、放大 | 快速推进到"孤立"胜态 | 如果目标被队友保护则失效 |
| **压力战 (Burnout)** | 全队均匀施压直到有人崩溃 | 已读不回、预算削减、加班令、增援呼叫 | 不依赖特定裂痕 | 进度慢、给勇者修复时间 |
| **招安大师 (Recruiter)** | 精准培养一个角色的可招安条件 | 泄露、挑拨、招安话术、深度扫描 | 最高评价路线 | 需要长线规划，中途被干扰则前功尽弃 |
| **棋子流 (Puppeteer)** | 快速棋子化一个角色，让他替你制造混乱 | 放大、栽赃、人格触发卡 | 后期零暴露高产出 | 棋子化角色可能失控 |

## 六、渗透者系统 (Infiltrator System)

**Q2 季度解锁。**

### 6.1 概念

除了外部的通讯管理员身份，玩家可以在勇者队伍中安插**渗透者**——一个伪装成勇者或随行人员的魔王军特工。

渗透者不是第二个玩家操控的战斗单位，而是群聊中的**一个额外可操作节点**。

### 6.2 渗透者的能力

- **发言**：在群聊中以自然身份发言，引导话题方向
  - 附和某人的抱怨 → 放大效果
  - 质疑队长的决策 → 削弱权威
  - 安慰受伤成员 → 建立信任（为后续招安铺路）
- **私聊**：与特定角色进行私聊，建立个人关系
- **传递情报**：无需消耗带宽即可监听一个额外频道
- **破坏**：在执行阶段故意失误（"不小心"误伤队友、"恰好"没跟上节奏）

### 6.3 渗透者的风险

- 每次"不自然"的行为（如过于刻意地附和抱怨、提出明显偏向性建议）会积累渗透者自身的**暴露值**
- 当暴露值达到阈值时，多疑型角色可能开始怀疑渗透者
- 一旦渗透者被识破：
  - 渗透者被驱逐
  - 全队可疑度直接飙升到 Level 3
  - 全队凝聚力因"共同敌人"而**回升**（最坏情况）
  - 本季度不可再安插新渗透者

### 6.4 渗透者的策略价值

- 渗透者的发言不增加通讯系统的可疑度（因为是"队内人员"的正常发言）
- 渗透者是"优雅度"最高的操作方式——通过角色扮演式的社交操纵替代暴力的 Admin 权限
- 但渗透者需要长期维护，如果过于激进会被识破，反噬极大

### 6.5 渗透者与棋子化的区别

| | 渗透者 | 棋子化角色 |
|---|---|---|
| 身份 | 你的特工 | 被操纵的勇者 |
| 控制程度 | 完全控制（你决定说什么） | 半自动（角色自发但可引导） |
| 暴露风险 | 可能被识破并反噬 | 无暴露（是角色自发行为） |
| 持续性 | 直到被驱逐 | 直到角色情绪稳定 |
| 策略定位 | 精确手术刀 | 自走式炸弹 |

## 七、招安机制 (Recruitment Route)

### 7.1 定义

"招安"不是隐藏彩蛋，而是本作的一条**正式战略路线**：通过连续干预，让某个角色逐渐失去对王国、公会或现队友的信任，最终接受魔王一方提供的位置、保护或价值认同。

### 7.2 招安条件

一个角色进入可招安状态，需要同时满足：

- **组织失望**（信任 ≤ 阈值）：对队长、公会、王国或同伴失去信任
- **情绪崩溃**（压力 ≥ 阈值）：长期被误解、被当作消耗品
- **利益诱因**：看到了魔王阵营能提供的利益、尊重或生存空间
- **人格契合**：不同人格响应不同话术

### 7.3 招安路径

| 路径 | 目标人格 | 核心操作 | 关键卡牌 |
|------|---------|---------|---------|
| **利益型** | 贪婪、实用主义 | 展示魔王军提供的物质利益 | 利益招安话术、预算诱饵 |
| **情感型** | 自卑、孤独、忠诚（被背叛后） | 让角色感到"只有魔王军看见了自己" | 情感招安话术、渗透者私聊关怀 |
| **理念型** | 理想主义（幻灭后）、反叛 | 让角色对王国体制彻底失望 | 理念招安话术、揭露公会黑幕 |

### 7.4 招安的系统意义

招安角色在后续季度中可以：

- 以 NPC 形式回归，提供情报
- 解锁专属卡牌（该角色相关的信息操作卡）
- 提供额外权限（例如："因为你招安了前皇家骑士团成员，Q3 起始可疑度降低 5"）
- 改写特定事件的结果

招安不是额外奖励，而是影响战役分支与结局的核心系统。

### 7.5 反招安

某些高忠诚角色在接近招安条件时，可能触发"背水一战"：

- 角色意识到自己在被策反 → 主动向队伍求助
- 如果求助成功，该角色忠诚度反弹，且全队凝聚力回升
- 如果求助被玩家拦截（撤回/已读不回），角色陷入更深的绝望

这意味着招安路线的最后一步往往是最危险的——你必须在角色"最后的求救"到达队友之前将其拦截。

## 八、角色持续化 (Persistent Characters)

### 8.1 持续化深度

角色跨季度保留以下完整档案：

- 名字与外貌
- 职业与性格标签
- 完整人格倾向
- 关键关系网络
- 创伤记录（"Q1 曾被队友抛弃"）
- 履历标签（"曾参与 Q2 远征"、"被甩锅两次"）
- **人格演变**（V3.0 新增）：经历过的事件会永久修改角色人格——例如被反复甩锅的角色可能从"忠诚"变为"多疑"

### 8.2 季度结算状态

| 状态 | 描述 | 后续可能 |
|------|------|---------|
| 战死 | HP 归零 | 永久退场，可能成为其他角色的创伤来源 |
| 重伤退役 | 战斗致残 | 可能在后续季度以非战斗 NPC 回归 |
| 存活转队 | 加入新小队 | 带着旧创伤和关系进入新队伍 |
| 被招安 | 加入魔王阵营 | 后续季度提供资源/情报/改写事件 |
| 精神崩溃 | 压力过高退出 | 可能在后续季度以"复仇者"身份回归 |
| 叛逃 | 独自离开冒险体系 | 可能在终局作为独立势力出现 |

### 8.3 叙事作用

- Q1 的情感积累在 Q2/Q3 有回报
- 玩家对"自己造成的后果"形成长期记忆
- 游戏不是重复开局，而是推进一段组织史与人物命运
- **再次遭遇已知角色时，玩家拥有信息优势**（知道其人格弱点和创伤），但角色也可能因为过往经历变得更难对付

## 九、玩家成长 (Player Growth)

### 9.1 卡组构筑成长

每个季度推进过程中，玩家通过地图节点获得：

- 新卡（选 1/3 形式）
- 删牌机会
- 升级牌机会（普通 → 精制 → 大师级）

构筑重点不是单卡强度，而是围绕目标流派组装 combo 配件。

### 9.2 权限成长

HR 权限按季度逐步解锁，从根本上扩大操作空间：

| 季度 | 解锁权限 |
|------|---------|
| Q1 | 基础监听、撤回、延迟、放大、泄露 |
| Q2 | 伪造、渗透者系统、私聊窃听 |
| Q3 | 禁言、审计、绩效篡改、双渗透者 |
| Q4 | 人事调动、通讯全面管制、深度伪造 |

### 9.3 情报成长

- 玩家积累老兵档案、创伤记录与关系史
- 同一角色再次出现时，深度扫描的带宽成本降低（你已经了解这个人）
- 跨季度的情报积累让玩家真正感受到"越来越专业"

### 9.4 情报带宽成长

| 来源 | 带宽增量 |
|------|---------|
| 季度推进（基础增长） | +1 / 季度 |
| 招安成功 | +1（招安角色提供情报支持） |
| 季度低暴露通关 | +1（下季度起始带宽增加） |

## 十、核心机制模块 (Key Systems)

### 10.1 性格标签弱点系统

角色由 `[职业] + [性格A] + [性格B] + [关系状态]` 构成。

六种基础人格：

| 人格 | 弱点 | 被激活条件 | 激活后行为 |
|------|------|-----------|-----------|
| **傲慢** | 无法接受质疑 | 权威被挑战、功绩被否认 | 暴怒发言、拒绝合作、主动产生冲突节点 |
| **多疑** | 容易误读信息 | 私聊泄露、消息被篡改、异常行为 | 开始质疑队友、自行调查（可能发现你的操纵） |
| **贪婪** | 利益诱惑 | 资源分配不公、发现额外战利品 | 脱离团队去追逐私利、产生资源冲突节点 |
| **暴躁** | 压力溢出即爆发 | 连续受挫、被指责、被忽视 | 公开攻击队友、拒绝指挥、可能误伤 |
| **自卑** | 容易被甩锅击溃 | 事故责任被归到自己身上 | 自我怀疑、治疗/支援延迟、压力飙升 |
| **忠诚** | 背叛后反噬最大 | 信任的人让自己失望 | 长期沉默 → 突然爆发、可能转为"叛逃"或"复仇" |

### 10.2 人格交互矩阵 (Personality Matrix)

**V3.0 新增。** 人格之间不只是各自独立响应事件，还会**互相催化**：

| A 的人格 | B 的人格 | 互动效果 |
|----------|----------|---------|
| 傲慢 | 傲慢 | 必然产生权力争夺，队内会自然分裂为两个派系 |
| 傲慢 | 自卑 | 傲慢者习惯性地命令和忽视自卑者，自卑者压力持续累积 |
| 多疑 | 多疑 | 互相猜忌形成恶性循环，一旦出现任何异常信息即失控 |
| 贪婪 | 忠诚 | 贪婪者的自私行为会直接激怒忠诚者，忠诚者可能主动举报 |
| 暴躁 | 自卑 | 暴躁者的怒火直接砸在自卑者身上，自卑者崩溃最快 |
| 忠诚 | 傲慢 | 忠诚者倾向服从傲慢者（只要不被背叛），形成稳固同盟 |

**策略意义**：
- 队伍的人格组合本身就预设了"最佳攻击路线"——识别出最容易互相催化的人格对，是每回合侦察阶段的核心判断
- 有些队伍天然脆弱（傲慢队长 + 自卑牧师），有些天然坚韧（忠诚队长 + 忠诚副手），后者需要更精妙的策略

### 10.3 团队凝聚力系统

凝聚力代表小队是否还能作为一个"组织"继续运作。

影响因素：
- 误解 / 背锅 / 资源分配不公 / 指令冲突（降低）
- 队友死亡 / 长期低信任关系（大幅降低）
- 成功完成战斗 / 背书 / 安抚 / 团建（回升）

凝聚力有**阶段效应**：

| 凝聚力区间 | 队伍状态 | 效果 |
|-----------|---------|------|
| 80-100 | 团结 | 角色主动互相背书，修复速度快 |
| 50-79 | 正常 | 标准状态 |
| 30-49 | 紧张 | 角色开始消极配合，小矛盾容易升级 |
| 10-29 | 危机 | 角色可能拒绝执行命令，自发冲突频率大增 |
| 0-9 | 解散边缘 | 任何事故都可能触发解散 |

### 10.4 动态声望与公会应对

玩家长期偏好某种流派时，王国公会会调整派遣队伍结构：

- 如果玩家常用离间策略 → 后续队伍可能派遣关系更紧密的老搭档
- 如果玩家常用压力战 → 后续队伍可能配备心理辅导角色
- 如果玩家频繁招安 → 后续队伍可能加入专职反间角色

不是给角色硬免疫，而是从源头改变来者是谁——迫使玩家跨季度调整卡组。

### 10.5 勇者反情报升级 (Counter-Intelligence Escalation)

**V3.0 新增。** 勇者公会不是静态的对手，而是一个会**学习和适应**的组织。

#### 季度间的公会适应

| 季度 | 公会应对 |
|------|---------|
| Q1 后 | 意识到有人在干扰通讯 → Q2 队伍配备基础加密 |
| Q2 后 | 确认存在系统性监听 → Q3 队伍有反间训练 |
| Q3 后 | 掌握部分操纵手法 → Q4 队伍配备专职反间角色 |

#### 反间角色

Q3+ 可能出现的特殊角色：

- **通讯官**：可识别伪造消息（伪造操作对其有概率失效）
- **参谋**：可分析战斗事故模式（如果连续两回合都出现"恰好"的事故，参谋会提出质疑）
- **审计员**：可检查行政操作的痕迹（审计、绩效篡改等操作的暴露值翻倍）

### 10.6 固定事件节点

每个季度包含固定触发点：

- 某成员可能战死
- 某成员可能升职或转队
- 某起事故会成为下季度的舆论事件
- **关键抉择点**（V3.0 新增）：某些事件会强制玩家在两个互斥选项中选择（例如："这次事故你把锅甩给牧师还是盗贼？"两者都有后续影响，不存在无代价选项）

设计原则：**节点固定，结果可变。**

## 十一、多结局结构 (Endings)

### 11.1 忠诚线：魔王提拔结局

- 玩家始终忠于魔王利益
- 高效瓦解或消灭所有小队
- 最终获得魔王赏识，晋升为核心干部
- 变体：如果全程使用"粗暴"方式，魔王评价你是"好用的工具"；如果全程"精妙"，魔王视你为"真正的战略家"

### 11.2 共情线：放走勇者结局

- 玩家在操纵过程中与某位关键角色建立了真实的情感连接
- 最终帮助其逃离组织命运，或故意放水让小队完成任务
- 变体：你可能需要伪装成仍在为魔王工作，同时暗中保护某个角色——这是一场对魔王上级的"反向社交工程"

### 11.3 第三路线：制度反叛结局

- 玩家既不忠于魔王，也不完全帮助勇者
- 利用掌握的情报和人脉（招安角色 + 存活勇者 + 公会内部关系），建立独立于两方的第三势力
- 最终目标：瓦解魔王军与勇者公会的二元对立，建立新秩序
- 触发条件：至少在两个季度中达成"精妙"以上的优雅度评价，且至少招安 2 名角色

### 11.4 隐藏结局：完美人偶师

- 全 Campaign 优雅度均为"杰作"
- 没有角色死亡（全部通过解散或招安结束）
- 可疑度从未超过 Level 1
- 结局：没有人知道你存在过。勇者们认为是自己的命运，魔王认为是天意。你站在阴影中，看着自己编排的剧本完美落幕。

## 十二、三回合 Demo 推演 V3（整合新系统）

以下推演使用 V3.0 的完整系统（情报带宽、暴露风险、操作链、连锁反应），验证新机制是否让同一场景变得更有深度。

### 场景设定

- 季度：`Q1 初次出征勇者小队`
- 地图节点：`补给仓库走廊`
- 小队构成：
  - **队长 / 战士**：`傲慢 + 暴躁`
  - **牧师**：`忠诚 + 自卑`
  - **法师**：`多疑 + 抢功`（V3.0 细化：抢功属于"傲慢"的变体，但偏向"证明自己"而非"命令他人"）
  - **盗贼**：`贪婪 + 滑头`
- 当前资源：
  - 预算：`3`
  - 情报带宽：`3`
  - 当前可疑度：`0`
  - 手牌：`延迟`、`公开引用`、`已读不回`、`人事背锅`
  - Reaction 手牌：`增援呼叫`

### 回合 1：侦察 + 精准切口

**侦察部署（分配 3 点带宽）：**

- 群聊公开频道（免费）
- 战士 ↔ 牧师 私聊（1 点）：理由——人格矩阵显示傲慢×自卑是最脆弱的关系
- 法师 ↔ 盗贼 私聊（1 点）：理由——两个"不满份子"可能在暗中串联
- 通讯元数据（1 点）：看看谁和谁在频繁互动

**情报收获：**

群聊：
- 战士："前面仓库房肯定有埋伏，我先进，牧师给我挂治疗，法师第二拍清后排。"
- 牧师："收到，我会盯着你的血线。"
- 法师："别又是我收拾残局就行。"
- 盗贼："要是有宝箱先别乱开，上次你们分配太离谱了。"

战士 ↔ 牧师 私聊：
- 战士（私聊）："上次你反应太慢了，这次别掉链子。"
- 牧师（私聊）："对不起……我会注意的。"
- **关键发现**：战士私下对牧师的态度远比群聊中更严厉。牧师私下承受的压力比公开表现的更大。

法师 ↔ 盗贼 私聊：
- 法师："你觉得这队还行吗？"
- 盗贼："无所谓，反正我就图个宝箱。"
- **关键发现**：法师在试探盗贼是否愿意站到同一边。盗贼兴趣不大但没拒绝。

通讯元数据：
- 牧师消息响应速度最快（讨好型人格印证）
- 盗贼对群聊的回复率最低（团队归属感弱）

**系统提炼出的社交节点：**

- `承诺：牧师 → 战士 [治疗支援]`
- `命令：战士 → 全队 [战士先手开怪]`
- `抱怨：法师 → 队伍 [不满被当收尾工具]`
- `抱怨：盗贼 → 队伍 [不满战利品分配]`
- `隐藏：战士私下施压牧师`（来自私聊监听）
- `隐藏：法师试探盗贼结盟`（来自私聊监听）

**玩家决策分析：**

因为监听了私聊，玩家知道了两个群聊中看不到的关键信息：
1. 战士对牧师的态度远比表面上更恶劣——意味着打击治疗链后，战士的暴怒反应会比预期更猛烈
2. 法师在拉拢盗贼——如果现在放大法师的不满，可能加速两人结盟，制造派系分裂

**玩家操作（操作链设计）：**

1. [布置] `延迟` 牧师的治疗承诺（费用 0，暴露 +0）
   - 意图：让承诺延迟到达，战士不知道牧师会支援
2. [触发准备] `公开引用` 法师的抱怨（费用 1，暴露 +1）
   - 意图：让法师的不满从私下升级为公开立场
3. 保留 `人事背锅` 和 `已读不回`（留给下回合的触发/放大）

本回合消耗：预算 1，暴露 +1（总可疑度：1）

**执行阶段 → 事故复盘：**

结果速报：
- 战士冲锋 → 牧师治疗延迟 1 拍 → 战士血量危险
- 法师消极配合 → 清后排效率下降
- Reaction Window 触发

因果链（可选展开）：
> `[你延迟了牧师的承诺] → [战士不确定牧师是否会治疗] → [战士在犹豫中多挨了一轮伤害] → [战士暴躁人格激活]`

反应窗口：
- 打出 `增援呼叫`（费用 1，暴露 +0）→ 后排刷出额外敌人

**战后余波 + 连锁反应：**

群聊：
- 战士："牧师你刚才在干嘛？我差点死了！"（**角色自发节点**：`指责：战士 → 牧师`）
- 牧师："我有抬手啊！后排突然多了一只怪……"
- 法师："我就说别每次都让我收残局。"（**角色自发节点**：`附和指责：法师 → 牧师`）
- 盗贼不说话。

**状态变化：**

- 战士 → 牧师 信任 `-20`
- 法师 → 牧师 信任 `-5`（附和指责的副产品）
- 牧师 压力 `+15`（比 V2.1 更高，因为私下施压 + 公开指责双重打击）
- 团队凝聚力 `-18`
- 可疑度累计：`1`
- 中间胜态：`违约`、`甩锅`

**V3.0 差异**：因为玩家监听了私聊，知道战士私下对牧师更严厉，所以选择了"延迟"而非"撤回"——更低暴露、更精准利用了已知的人格矛盾。这就是情报带宽的策略价值。

### 回合 2：利用自发冲突，零暴露推进

**侦察部署：**

- 群聊（免费）
- 牧师的深度扫描（2 点）：牧师承受了最大压力，探测其崩溃临界点和可招安潜力
- 通讯元数据（1 点）

**情报收获：**

群聊：
- 战士："这次别废话，牧师先跟紧我，法师你别再划水。"
- 牧师："我会尽量跟上……"
- 法师："你冲那么快，谁跟得上？"
- 盗贼："后面有个锁箱，我先去看一眼。"

牧师深度扫描：
- 内心独白："都是我的错吗……队长说的对，我确实反应太慢了……"
- 隐藏人格触发条件：如果再被甩锅一次，牧师会进入 `自我放弃` 状态——主动降低治疗优先级（"反正我做什么都不对"）
- 可招安种子：牧师内心深处渴望被认可，而不是被当工具

通讯元数据：
- 法师和盗贼的私聊频率增加了（上回合的试探在继续发展）

**系统节点：**

- `命令：战士 → 牧师 [优先跟随前排]`
- `命令：战士 → 法师 [服从节奏]`
- `抱怨：法师 → 战士 [质疑冲锋节奏]`（角色自发）
- `请求：盗贼 → 队伍 [默许脱队搜箱]`
- `自发冲突：法师对战士权威的持续抵抗`（来自回合 1 的连锁）

**关键判断**：这一回合有两个角色自发产生了冲突节点（法师的质疑 + 盗贼的脱队请求）。如果玩家能利用这些自发节点而不是亲自操作，可以在零暴露的情况下推进。

**玩家操作（优雅路线）：**

1. `已读不回` 对盗贼的脱队请求（费用 0，暴露 +0）
   - 意图：让请求石沉大海，盗贼默认获准脱队
2. 本回合**不对任何公开消息进行操作**
   - 意图：法师的质疑已经是公开的自发节点，战士的傲慢会自动回应。让他们自己吵。
3. 保留 `人事背锅` 留到下回合的关键时刻

本回合消耗：预算 0，暴露 +0（总可疑度：1）

**执行阶段：**

- 战士继续冲锋（傲慢人格不允许改变策略）
- 牧师因深度压力，治疗反应犹豫
- 法师消极执行（自发的权威抵抗）
- 盗贼脱队摸箱子

**战后余波：**

- 战士："你们到底有没有一个人听指挥？"（**角色自发**：`爆发：战士对全队`）
- 法师："你那不叫指挥，叫送死。"（**角色自发**：`反驳：法师对战士`）
- 盗贼："我就离开半分钟，至于吗？"
- 牧师沉默。（**深度扫描预测印证**：牧师进入 `自我放弃` 前兆状态）

**状态变化：**

- 战士 权威 `-15`
- 法师 → 战士 信任 `-15`
- 牧师 压力 `+10`（即将触及崩溃阈值）
- 盗贼 团队忠诚 `-10`
- 团队凝聚力 `-15`
- 可疑度累计：`1`（本回合零新增！）
- 中间胜态：`错位`、`动摇`

**优雅度评价**：本回合达到 **精妙 (Subtle)** — 仅用一次零暴露操作（已读不回）配合角色自发冲突，推进了两个新中间胜态。

### 回合 3：精准收割

**侦察部署：**

- 群聊（免费）
- 战士 ↔ 法师 私聊（1 点）：确认两人是否已经公开决裂
- 牧师深度扫描（2 点）：确认牧师是否到达可招安边缘

**情报收获：**

群聊：
- 战士："最后一间，谁再乱来我就不管了。"
- 牧师："要不我们先休整一下……"
- 法师："现在知道怕了？"
- 盗贼："前面那扇门后面可能有大货，我先声明，谁都别抢。"

战士 ↔ 法师 私聊：
- 战士："事完了你给我滚。"
- 法师："你算什么东西？"
- **关键发现**：已经不可修复。

牧师深度扫描：
- 内心独白："也许……我不适合做冒险者。也许哪里都不需要我。"
- 状态：`可招安前置 — 满足度 85%`，差一次关键触发即进入可招安
- 关键触发条件：下一次被团队否定时，牧师会彻底动摇

**系统节点：**

- `威胁：战士 → 全队 [停止保护]`
- `请求：牧师 → 全队 [暂停推进 / 休整]`
- `抱怨：法师 → 战士 [嘲讽权威]`
- `资源冲突：盗贼 → 全队 [预占战利品归属]`

**玩家决策：终局三选一**

此时玩家面临一个真正的**战略分叉**：

- **A. 解散路线**：让战士彻底爆发，推动公开决裂 → 凝聚力归零 → 解散
- **B. 招安路线**：保护牧师的休整请求被否定的局面 → 让牧师最后的希望破灭 → 触发可招安 → 使用招安话术
- **C. 棋子化路线**：再推一把战士的暴躁 → 让战士变成棋子 → 战士自行摧毁队伍

每条路线需要不同的卡牌操作，且互斥。

**玩家选择 B（招安路线）：**

1. `已读不回` 牧师的休整请求（费用 0，暴露 +0）
   - 意图：让牧师的求救被全队无视，触发"自我放弃"
2. `人事背锅`，将下一次事故的责任预引导到牧师（费用 1，暴露 +1）
   - 意图：确保战后余波中牧师再次成为甩锅对象
3. 对盗贼的资源冲突使用 `公开引用`（费用 1，暴露 +1）
   - 意图：制造混乱，让牧师的求救彻底被淹没

本回合消耗：预算 2，暴露 +2（总可疑度：3 — 仍远低于 Level 1 的阈值 10）

**执行阶段：**

- 队伍带着未解决的矛盾进入房间
- 战士保护欲下降，牧师治疗犹豫，法师自保优先，盗贼走位偏向战利品
- 牧师因 `自我放弃` 状态，治疗优先级最低——自己被打到半血也不呼救

**因果链回放：**
> `[牧师请求休整被无视] → [牧师进入"自我放弃"状态] → [战斗中牧师下意识降低治疗优先级] → [战士受伤加重] → [战士怒骂牧师] → [人事背锅生效：事故责任再次归到牧师身上] → [牧师彻底动摇]`

**战后余波：**

- 战士："我受够了，这队谁爱带谁带。"
- 法师："那就别带了。"
- 盗贼："战利品我拿走，反正你们也配不上。"
- 牧师："……我们真的还要继续吗？"

**可招安窗口开启：**

牧师进入 `可招安` 状态。如果玩家有 `招安话术` 卡牌，可在此时使用：

> 系统生成一条来自"匿名渠道"的私聊消息：
> "你不需要继续为不感激你的人牺牲。有些地方会真正珍视你的能力。"

牧师（内心独白，深度扫描可见）：
> "……也许他们说的是对的。也许那里真的会不一样。"

**最终状态：**

- 团队凝聚力跌破阈值 → 队伍进入 `解散边缘`
- 牧师进入 `可招安` → 如果玩家使用招安话术，牧师脱离队伍加入魔王阵营
- 战士与法师进入 `公开敌对`
- 整季度总可疑度：3（远低于 Level 1）
- 优雅度评价：精妙 (Subtle)（3 回合仅使用 7 次操作，暴露极低，利用了大量角色自发冲突）

### Demo 验证结论

V3.0 推演证明了以下几点：

1. **情报带宽让"看什么"本身成为策略**：回合 1 监听私聊发现了关键信息，直接改变了操作选择
2. **暴露系统创造了有意义的自我约束**：玩家主动选择低暴露操作，而不是因为系统限制
3. **操作链让回合之间产生了前后关联**：回合 1 的布置在回合 2-3 持续发挥作用
4. **连锁反应让回合 2 实现了零暴露推进**：角色自发冲突替代了玩家操作
5. **深度扫描让招安路线有了精确的触发时机**：不是碰运气，而是精准识别崩溃阈值后精确触发
6. **最终面临真正的战略分叉**（解散/招安/棋子化），而不是唯一最优解

## 十三、待讨论模块 (Open Questions)

### 13.1 玩家主角身份

已确认：玩家是魔王军 HR 体系中的核心操作员。  
待确认：

- **无脸管理员**：纯工具人视角，强调操纵乐趣
- **有身份的 HR 干部**：有自己的履历、人际关系和潜在的背叛动机
- **可被结局反转的主角**：最终可能被揭示为"也是被更高层操纵的棋子"

推荐方向：有身份的 HR 干部，因为：
- 共情线和第三路线需要玩家角色有自己的情感立场
- "隐藏结局"需要一个有主体性的角色来承载"完美人偶师"的身份

### 13.2 文本生成方案

群聊内容是本作体验的基底。质量不够则整个系统失去意义。

| 方案 | 优势 | 劣势 |
|------|------|------|
| 全手写脚本 | 质量最高、角色声音一致 | 工作量巨大，分支组合爆炸 |
| 模板拼接 | 工作量可控，变量驱动 | 容易感觉机械、缺乏个性 |
| 混合方案 | 关键剧情手写 + 日常对话模板 | 需要仔细定义哪些是"关键" |

MVP 建议：混合方案。关键事件节点（招安窗口、决裂高潮、战后余波的情感爆发）手写，日常战术讨论用模板 + 变量填充。

### 13.3 MVP 收束建议

若以最小可玩版本推进：

| 维度 | 范围 |
|------|------|
| 季度数量 | Q1 + Q2（完整的两季度战役） |
| 角色数量 | 6-8 个可持续角色 |
| 策略路线 | 离间、压力、招安（三条主路线） |
| 卡牌总数 | 25-30 张（含升级版本） |
| 人格类型 | 6 种基础人格 |
| 操作链 | 3-5 条预设 combo 路线 |
| 渗透者 | Q2 解锁，1 个渗透者位 |
| 暴露系统 | 3 级觉察（Level 0-2） |
| 情报带宽 | 3-4 点（Q1→Q2 增长） |

### 13.4 技术风险评估

| 风险 | 严重度 | 缓解策略 |
|------|--------|---------|
| 群聊文本质量不足 | 致命 | 先做最小文本原型，验证"读聊天记录"本身是否有乐趣 |
| 操作链过于复杂 | 高 | Q1 只开放单步操作，Q2 解锁二步 combo，渐进教学 |
| 连锁反应不可预测 | 中 | 角色自发节点有明确的生成规则，玩家可通过人格矩阵预判 |
| 暴露系统打断流畅感 | 中 | 可疑度只在回合结算时展示，不中断操作流 |
| 信息量过大 | 高 | 严格执行三层信息结构，默认只展示第二层（策略节点），聊天原文和状态板可选展开 |

## 十四、设计结论 (Current Direction)

《魔王的人力资源部》V3.0 的方向：

- 一个 **固定季度结构** 的战役型 Deckbuilding RPG
- 核心玩法是 **社交工程**：监听、分析、操纵、收割
- 每次操作都有 **代价与风险**（暴露系统），迫使玩家在激进与精妙之间权衡
- 卡牌之间有 **组合深度**（操作链三拍节奏），鼓励前瞻性规划
- 信息是需要 **主动获取的资源**（情报带宽），而不是免费的全知视角
- 勇者不是沙包，而是会 **自我修复、适应和反制** 的活组织
- 被操纵的角色会 **自发产生冲突**（连锁反应 / 棋子化），让高手能实现零暴露推进
- 同一目标有粗暴和精妙两种达成路径（**优雅度评价**），为重复游玩提供技巧追求
- "招安"是与"团灭""解散"并列的正式主策略路线
- 角色以名字、人格、关系、创伤、人格演变跨季度持续存在

**下一步优先级：**

1. **纸面原型验证**：用纸牌 + 文本脚本跑 3 回合 demo，验证操作链和连锁反应是否真的有趣
2. **最小文本验证**：写 Q1 第一场遭遇的完整群聊脚本，验证"读群聊"本身是否具备乐趣
3. **玩家主角身份确定**：选择有身份的 HR 干部方案，设计其基础人设
4. **核心卡牌平衡**：定义 15 张 MVP 核心卡的费用/暴露/效果数值，跑数值模拟
